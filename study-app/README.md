# אינפי - מערכת לימוד אינטליגנטית למבחן

מערכת לימוד חכמה ומתקדמת שעוזרת לך להתכונן למבחן האינפי (Calculus/Infinitesimal) על ידי ניתוח הרצאות, תרגולים, תרגילי בית ומבחנים קודמים.

## 🎯 תכונות עיקריות

### 1. ספריית קבצים + העלאה חכמה
- העלאת מספר קבצי PDF בו-זמנית
- זיהוי אוטומטי של סוג הקובץ (הרצאה/תרגול/תרגיל בית/מבחן)
- חילוץ טקסט מקבצי PDF באמצעות PDF.js
- תצוגת PDF מובנית
- ניהול מלא (מחיקה, סינון, חיפוש)

### 2. בסיס ידע - תוכן VERBATIM
- חילוץ אוטומטי של הגדרות, משפטים, למות, מסקנות, הוכחות וטכניקות
- **תוכן מדויק מהמקור** - ללא שינוי ניסוח
- קישור ישיר למקור (קובץ + עמוד)
- תיוג לפי נושאים (גבולות, רציפות, נגזרות, אינטגרלים, טורים)
- סינון חכם לפי סוג, נושא וסבירות

### 3. מנוע סבירות למבחן
- ניתוח מבוסס-נתונים של מה חשוב ביותר
- דירוג משפטים, הוכחות והגדרות לפי תדירות במבחנים
- זיהוי פערים ונקודות לשיפור
- המלצות מותאמות אישית

### 4. חיזוי תרגילי בית למבחן ⭐
- **חשוב**: 2 שאלות במבחן יהיו משאלות התרגילים!
- דירוג כל שאלות התרגילים לפי סבירות (70%+ / 40%-69% / <40%)
- זיהוי דפוסים משותפים עם מבחנים קודמים
- קישור למשפטים רלוונטיים

### 5. ניתוח מבחנים קודמים
- מבנה המבחן הטיפוסי
- נושאים ומשפטים שכיחים
- זיהוי דפוסי שאלות
- התפלגות קושי

### 6. מצב תרגול אינטראקטיבי
- **כרטיסיות**: תרגול הגדרות ומשפטים
- **זיהוי כלי**: תרגול על "מתי להשתמש"
- מעקב אחר תשובות ושיפור
- מיקוד בפריטים בעלי סבירות גבוהה

### 7. תכנית לימוד מותאמת אישית
- יצירת תכנית אוטומטית לפי תאריך המבחן
- תכנית יומית מפורטת עם משימות
- מעקב התקדמות
- Spaced Repetition

### 8. דשבורד מרכזי
- סטטיסטיקות מלאות
- רצף לימוד (Study Streak)
- פעולות מהירות
- תכנית היום

## 🚀 התקנה והרצה

### דרישות מקדימות
- Node.js 18+ מותקן במחשב

### צעדי התקנה

```bash
# התקן תלויות
npm install

# הרץ שרת פיתוח
npm run dev

# פתח בדפדפן
# http://localhost:3000
```

### בניה לפרודקשן

```bash
npm run build
npm start
```

### **שלבי הפעלה:**

2. **פתח ניטור מקורות** ב-http://localhost:3000/library
3. **לחץ "סרוק מחדש"** לזיהוי קבצים
4. **לחץ "עבד את כל הקבצים"** לחילוץ תוכן
5. **חקור** את בסיס הידע, מנוע הסבירות, חיזוי תרגילים
6. **צור תכנית לימוד** והתחל לתרגל

## ✨ עקרונות חשובים

- **תוכן VERBATIM**: כל התוכן המדויק מהמקור נשאר ללא שינוי
- **ניתוח מבוסס-נתונים**: דירוגים על בסיס תדירות במבחנים ודמיון לשאלות
- **אחסון מקומי**: כל המידע נשמר ב-IndexedDB בדפדפן (פרטיות מלאה)
- **RTL ועברית**: תמיכה מלאה בעברית וכיוון ימין-שמאל
- **Dark Mode**: מצב כהה נוח לעיניים

## 📝 הערות

- **PDF מורכבים**: חילוץ טקסט מקבצים עם נוסחאות מורכבות עשוי להיות לא מושלם
- **ניתוח מדויק**: ככל שתעלה יותר מבחנים קודמים, הניתוח ישתפר
- **ביצועים**: כל העיבוד נעשה בדפדפן - קבצים גדולים (<50MB) מומלצים
