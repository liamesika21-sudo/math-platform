# מטלה 1

## שאלה 1 (6 נקודות)
תנו הגדרה מתמטית מדויקת לכל אחד מהמושגים הבאים. אם ההגדרה שלכם משתמשת במושג שנלמד בקורס, הגדירו גם אותו. אין להשתמש במילת שלילה או סימן שלילה, למעט אולי הסימן ≠.

**(א)** (2 נקודות) יהי 1 < p ∈ ℕ. הגדירו את המושג: p הוא מספר פריק (composite).

**(ב)** (2 נקודות) יהי n ∈ ℤ. הגדירו את המושג: 2 ∤ (2n − 1).

**(ג)** (2 נקודות) יהיו n, m ∈ ℤ. הגדירו את המושג: n ו-m אינם זרים (not coprime).

---

## שאלה 2 (12 נקודות)
לכל אחת מהטענות הבאות, כתבו את השלילה הלוגית שלה **ללא** שימוש במילת שלילה או סימן שלילה מפורשים, למעט אולי הסימן ≠. אין צורך לקבוע את ערך האמת של הטענה המקורית.

**(א)** (3 נקודות)
לכל q ∈ ℚ קיים r ∈ ℚ כך ש-q < r < q + 1.

**(ב)** (4 נקודות)
קיים M > 0 כך שלכל n ∈ ℕ עבורו n > M מתקיים:
$$\left|\frac{n + 2}{n} - 1\right| < \frac{1}{n}$$

**(ג)** (5 נקודות)
לכל x ∈ ℝ ולכל ε > 0 קיים δ > 0 כך שלכל y ∈ ℝ עבורו 0 < |y − x| < δ מתקיים |y² − x²| < ε.

---

## שאלה 3 (12 נקודות)
יהיו a, b ∈ ℤ.

**(א)** (7 נקודות) נניח ש-a, b זרים. הוכיחו שלכל k ∈ ℕ מתקיים ש-a ו-bᵏ זרים.

הערה: ניתן להניח שלכל n ∈ ℕ כך ש-n > 1, קיים מספר ראשוני p > 1 כך ש-p | n.

**(ב)** (5 נקודות) יהיו x, y ∈ ℤ. נניח ש-ax + by = 1. הוכיחו ש-a, b זרים וש-x, y זרים.

---

## שאלה 4 (20 נקודות)
יהיו k, m, n ∈ ℕ. נניח ש-n, m זרים.

**(א)** (10 נקודות) יהיו a₀, ..., aₖ ∈ ℤ ונתבונן בפולינום המוגדר ע״י p(x) = a₀ + a₁x + ⋯ + aₖxᵏ לכל x ∈ ℝ. נניח ש-p(n/m) = 0. הוכיחו ש-m | aₖ.

**(ב)** (8 נקודות) נניח ש-ᵏ√n ∉ ℕ. הוכיחו ש-ᵏ√n הוא אי-רציונלי.

רמז: התבוננו בפולינום המוגדר ע״י p(x) = xᵏ − n לכל x ∈ ℝ.

**(ג)** (2 נקודות) הסיקו ש-∛14 הוא אי-רציונלי.

---

## שאלה 5 (36 נקודות)
הוכיחו או הפריכו כל אחת מהטענות הבאות.

**(א)** (3 נקודות) יהי 0 ≠ q ∈ ℚ ויהי r ∈ ℝ\ℚ. אז r · q ∈ ℝ\ℚ.

**(ב)** (3 נקודות) יהיו x, y ∈ ℝ\ℚ. אז x³y ∈ ℝ\ℚ.

**(ג)** (5 נקודות) יהיו x, y ∈ ℝ. נניח ש-x + y, x − y ∈ ℚ. אז x, y ∈ ℚ.

**(ד)** (7 נקודות) אם x, y ∈ ℝ\ℚ ו-x + y ∈ ℚ, אז x² + y² ∈ ℚ.

**(ה)** (8 נקודות) יהי 0 ≠ q ∈ ℚ ו-r ∈ ℝ\ℚ. אז:
$$\frac{r + q}{r - q} \in \mathbb{R} \setminus \mathbb{Q}$$

**(ו)** (10 נקודות) יהי x ∈ ℝ\ℚ כך ש-x ≥ 0. אז √(x + 1) − √x ∈ ℝ\ℚ.

---

## שאלה 6 (14 נקודות)
יהי x ∈ ℝ\ℚ ויהיו r₁, r₂, r₃, r₄ ∈ ℚ כאשר r₃x + r₄ ≠ 0. נתבונן במספר הממשי:
$$y = \frac{r_1 x + r_2}{r_3 x + r_4}$$

הוכיחו שאם r₁r₄ − r₂r₃ ≠ 0, אז y ∈ ℝ\ℚ.

# מטלה 2

## שאלה 1 (15 נקודות)
תהי ∅ ≠ A ⊆ ℝ.

**(א)** (5 נקודות) **(משפט החסם התחתון הגדול ביותר)** נניח ש-A חסומה מלמטה. הוכיחו שקיים חסם תחתון גדול ביותר יחיד של A, ̲s ∈ ℝ.

**(ב)** (5 נקודות) **(תכונת ה-ε של החסם התחתון הגדול ביותר)** נניח ש-A חסומה מלמטה ויהי ̲s ∈ ℝ חסם תחתון של A. הוכיחו ש-̲s = inf(A) אם ורק אם לכל ε > 0 קיים a ∈ A כך ש-a < ̲s + ε.

**(ג)** (5 נקודות) נניח ש-A חסומה מלמטה. הוכיחו ש-inf(A) ∈ A אם ורק אם ל-A יש ערך מינימלי.

---

## שאלה 2 (20 נקודות)
לכל אחד מהתנאים הבאים, תנו דוגמה לקבוצה ∅ ≠ A ⊆ ℝ המקיימת את התנאים.

הערה: אין צורך להוכיח את טענותיכם.

**(א)** (5 נקודות)
∀a ∈ A ∃b ∈ A : b < a וגם ∀M ∈ ℝ ∃a ∈ A : a > M.

**(ב)** (5 נקודות)
∀a ∈ A ∃b ∈ A : b > a וגם ∃M ∈ ℝ ∀a ∈ A : a ≤ M.

**(ג)** (5 נקודות)
∀m ∈ ℤ ∃a ∈ A : m < a < m + 1 וגם ∃x ∈ ℝ ∀a ∈ A : a ≤ x או a ≥ x + 1.

**(ד)** (5 נקודות)
∃x ∈ ℝ : x = inf(A) וגם ∀y ∈ ℝ מתקיים y ≠ sup(A).

---

## שאלה 3 (25 נקודות)

**(א)** (18 נקודות) לכל אחת מהקבוצות הבאות, חשבו את sup(A), inf(A), min(A) ו-max(A), או הוכיחו שגודל נתון אינו קיים:

  **i.** (9 נקודות)
  $$A = \left\{\frac{2x - 3}{3x + 5} : x \geq 0\right\}$$

  **ii.** (9 נקודות)
  $$A = \left\{\frac{3n + 2}{n + 4} : n \in \mathbb{N}\right\}$$

**(ב)** (7 נקודות) תהי ∅ ≠ A ⊆ ℝ ונתבונן בקבוצה B = {2a : a ∈ A}. נניח ש-sup(A) = ∞. הוכיחו ש-sup(B) = ∞.

---

## שאלה 4 (10 נקודות)

**(א)** (5 נקודות) סטודנט א׳ הוכיח ש-sup{x ∈ ℝ : x < 1} = 2 באופן הבא:

> נסמן A = {x ∈ ℝ : x < 1} ו-s̄ = 2. מספיק להוכיח ש-sup(A) = s̄.
> 1. יהי x ∈ A. אז x < 1 ≤ 2 = s̄ ולכן s̄ הוא חסם עליון של A.
> 2. נניח בשלילה שקיים חסם עליון M < s̄. מתכונת ה-ε, קיים x ∈ A כך ש-x > s̄ − |s̄ − M| = M. סתירה.

מצאו את הטעות של סטודנט א׳.

**(ב)** (5 נקודות) סטודנט א׳ הוכיח שהקבוצה A = {1/n : n ∈ ℕ} אינה חסומה מלמטה באופן הבא:

> נניח בשלילה ש-A חסומה מלמטה. אז לפי משפט החסם התחתון הגדול ביותר, קיים ̲s ∈ ℝ כך ש-̲s = inf(A). ברור שאין ל-A מינימום, ולכן ̲s < 1/n לכל n ∈ ℕ. מתכונת ה-ε של ̲s, אנו יודעים שלכל ε > 0 קיים a ∈ A כך ש-a < ̲s + ε. נבחר ε = 1/n − ̲s > 0. אז מתכונת ה-ε, נקבל:
> 1/n < ̲s + (1/n − ̲s) = 1/n
> סתירה.

מצאו את הטעות של סטודנט א׳.

---

## שאלה 5 (10 נקודות)
תהיינה ∅ ≠ A, B ⊆ ℝ. נניח ש-B חסומה ו-A ⊆ B. הוכיחו ש-A חסומה ושמתקיים:
$$\inf(B) \leq \inf(A) \leq \sup(A) \leq \sup(B)$$

---

## שאלה 6 (20 נקודות)
תהיינה ∅ ≠ A, B ⊆ ℝ חסומות. הוכיחו או הפריכו כל אחת מהטענות הבאות:

הערה: בשאלה זו, אין צורך להסביר מדוע sup(A), sup(B) ∈ ℝ.

**(א)** (5 נקודות) יהי t ∈ ℝ ונתבונן בקבוצה Aₜ = A ∩ (−∞, t]. נניח ש-Aₜ ≠ ∅. אז sup(Aₜ) = min{sup(A), t}.

**(ב)** (5 נקודות) יהי s̄ = sup(A). אם קיים r > 0 כך ש-(s̄ − r, s̄) ∩ A = ∅, אז ל-A יש ערך מקסימלי.

**(ג)** (5 נקודות) אם ∀a ∈ A ∃b ∈ B : a < b < sup(B), אז sup(A) < sup(B).

**(ד)** (5 נקודות) אם ∃b ∈ B ∀a ∈ A : a < b < sup(B), אז sup(A) < sup(B).

# מטלה 3

## שאלה 1 (6 נקודות)
תנו הגדרה מתמטית מדויקת לכל אחד מהמושגים הבאים. אם ההגדרה שלכם משתמשת במושג שנלמד בקורס, הגדירו גם אותו. אין להשתמש במילת שלילה או סימן שלילה, למעט אולי הסימן ≠.

תהי ∅ ≠ A ⊆ ℝ.

**(א)** (2 נקודות) A אינה חסומה מלמעלה.

**(ב)** (2 נקודות) A אינה חסומה.

**(ג)** (2 נקודות) ל-A אין ערך מקסימלי.

---

## שאלה 2 (20 נקודות)
סטודנטית X יצאה למסע האצילי של מציאת האינפימום של:
$$A = \left\{\frac{x + 2}{3} - \sqrt[3]{x} : x \geq 0\right\}$$

אחרי כמה "חישובים" היא אימתה ש:
$$x^{1/3} < \frac{x + 2}{3} \quad \forall x \geq 0$$
למעט אולי עבור x = 1. זה הוביל אותה להאמין ש-inf(A) = min(A) = 0. אז, הדים קלושים של קולו של המתרגל שלה חזרו: "סמכו על המתמטיקה, לא על האגודלים שלכם!". באנחה, סטודנטית X החליטה להוכיח את זה - כמו שצריך, הפעם.

**(א)** (5 נקודות) הוכיחו שלכל a, b ≥ 0 מתקיים:
$$\sqrt{ab} \leq \frac{a + b}{2}$$

**(ב)** (5 נקודות) הסיקו שלכל a, b, c, d ≥ 0 מתקיים:
$$\sqrt[4]{abcd} \leq \frac{a + b + c + d}{4}$$

**(ג)** (5 נקודות) הסיקו שלכל a, b, c ≥ 0 מתקיים:
$$\sqrt[3]{abc} \leq \frac{a + b + c}{3}$$

**(ד)** (5 נקודות) הסיקו ש-inf(A) = min(A) = 0.

## שאלה 3 (25 נקודות)

**(א)** (15 נקודות) נתבונן בקבוצה:
$$A = \left\{\frac{n \cdot m^2}{n^2 \cdot m + 1} : n, m \in \mathbb{N}\right\}$$

  **i.** (7 נקודות) הוכיחו ש-sup(A) = ∞.

  **ii.** (8 נקודות) הוכיחו ש-A חסומה מלמטה וחשבו את inf(A). קבעו האם ל-A יש ערך מינימלי.

**(ב)** (10 נקודות) נתבונן בקבוצה:
$$A = \left\{\frac{m + n + 1}{mn} : n, m \in \mathbb{N}\right\}$$

חשבו את sup(A) ו-inf(A), וקבעו האם ל-A יש ערך מינימלי והאם ל-A יש ערך מקסימלי.

---

## שאלה 4 (15 נקודות)
תהי ∅ ≠ A ⊆ ℝ חסומה, ונגדיר את **קבוצת נקודות האמצע** שלה:
$$M(A) = \left\{\frac{a_1 + a_2}{2} : a_1, a_2 \in A\right\}$$

הוכיחו ש:
$$\inf(M(A)) = \inf(A) \quad \text{וגם} \quad \sup(M(A)) = \sup(A)$$

## שאלה 5 (24 נקודות)
תהי ∅ ≠ A ⊆ ℝ. הוכיחו או הפריכו כל אחת מהטענות הבאות:

**(א)** (8 נקודות) יהיו x, y ∈ ℝ. אם לכל q ∈ ℚ כך ש-x < q מתקיים y < q, אז y ≤ x.

**(ב)** (8 נקודות) אם A צפופה ב-ℝ, אז A ∩ ℚ צפופה ב-ℝ או A ∩ (ℝ\ℚ) צפופה ב-ℝ.

**(ג)** (8 נקודות) A צפופה ב-ℝ אם ורק אם לכל x ∈ ℝ ולכל ε > 0 קיים a ∈ A כך ש-|a − x| < ε.

---

## שאלה 6 (10 נקודות)
תהי ∅ ≠ A ⊆ ℝ. נניח ש:

(i) A צפופה ב-ℝ.

(ii) לכל ∅ ≠ B ⊆ A כך ש-B חסומה מלמעלה, מתקיים sup(B) ∈ A.

הוכיחו ש-A = ℝ.
