## רקע

המערכת היא עוזר לימודי חכם המותאם לקורס ספציפי. הצ'אט מכוון על-ידי חומרי הקורס, מדיניות המרצה, ורמת הקושי הרצויה – ומאפשר לסטודנטים לקבל עזרה 24/7 בצורה שמשלימה את ההוראה בכיתה ולא מחליפה אותה.

הקורסים הראשונים להטמעה:
- **מבני נתונים**
- **לוגיקה ותורת הקבוצות**

|------|----------------|--------------------------|-------|
| 1    |                |                          |       |
| 2    |                |                          |       |
| ...  |                |                          |       |

**מטרה:** המערכת תדע מה הסטודנט **אמור לדעת** בכל נקודת זמן בסמסטר, ולא תתייחס לחומר שעוד לא נלמד.

### 2. חומרים שיש להעביר

- **מצגות / סיכומי הרצאות** (PDF, או כל פורמט)
- **גיליונות תרגול / שיעורי בית** (עם פתרונות אם אפשר)
- **נוסחאון / דפי עזר** שמורשים בבחינה

> כל חומר שתעביר יישאר בשימוש פנימי של המערכת בלבד ולא יופץ לסטודנטים ישירות.

---

### 3. כיוון הצ'אט – מדיניות עזרה

נבקשך לענות על השאלות הבאות כדי שנוכל לכוון את ההתנהגות של המערכת:

#### א. עד כמה הצ'אט יכול "לתת" תשובות?
- [ ] לרמוז בלבד (שאלות הנחיה, לא לתת את הפתרון)
- [ ] לתת פתרון חלקי ולבקש מהסטודנט להשלים
- [ ] לתת פתרון מלא עם הסבר
- [ ] תלוי בסוג השאלה – ציין להלן:

```
שאלות פתוחות / הוכחות: ___________
שאלות סגורות / חישוביות: ___________
שיעורי בית: ___________
```

#### ב. מה אסור לצ'אט לעשות?
דוגמאות אפשריות – ציין מה רלוונטי:
- [ ] לא לפתור שאלות שיעורי בית ישירות
- [ ] לא לתת פתרונות מבחנים עתידיים
- [ ] לא לתת "קיצורי דרך" שעוקפים את הלמידה
- [ ] אחר: ___________

#### ג. אוצר מילים ורמת הסבר
- האם להשתמש בסימון ספציפי של הקורס? (לדוג' שמות ספציפיים לאלגוריתמים, סימני לוגיקה מסוימים)
- רמת הסבר מועדפת: בסיסי / בינוני / מתקדם
- האם לתמוך בתשובות **בעברית בלבד**, **אנגלית בלבד**, או **שתיהן**?

### 4. רמות קושי לשאלות

כדי שהמערכת תוכל לספק לסטודנטים שאלות תרגול מותאמות, נבקש לסמן על השאלות הקיימות (או לספק מדגם):

| רמה | תיאור |
|-----|--------|
| ⭐ קל | הגדרות, בדיקת הבנה בסיסית |
| ⭐⭐ בינוני | שאלות יישום, תרגילים סטנדרטיים |
| ⭐⭐⭐ קשה | הוכחות, שאלות בסגנון מבחן |
| ⭐⭐⭐⭐ אתגרי | שאלות מבחן מועד ב', שאלות בונוס |

**חלופה:** אם אין זמן לסמן – ניתן לציין אילו **נושאים** נחשבים קשים במיוחד לסטודנטים.

### 5. מעקב וניתוח סטודנטים

המערכת יכולה לאסוף נתונים אנונימיים על שימוש. אנא ציין מה מעניין אותך:

- [ ] אילו נושאים הסטודנטים שואלים עליהם הכי הרבה
- [ ] אילו שאלות גורמות לקושי חוזר
- [ ] עד כמה מוקדם לפני המבחן מתחיל השימוש
- [ ] האם יש פערים בין קבוצות תרגול שונות
- [ ] דוחות שבועיים / חודשיים לפי נושא
- [ ] אחר: ___________

**תדירות דיווח מועדפת:**
- [ ] בזמן אמת (לינק לדשבורד)
- [ ] דוח שבועי במייל
- [ ] לפני כל מבחן
- [ ] לפי בקשה בלבד
