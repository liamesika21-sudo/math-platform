## 1. תחשיבי פסוקים (Propositional Logic)

### כללי זהב

**שקילויות קריטיות לשינון:**
- `p -> q  ≡  ¬p ∨ q` (זה הכלל הכי חשוב! מופיע כמעט בכל מבחן)
- `¬(p -> q) ≡ p ∧ ¬q` (שלילת גרירה - חוזר הרבה)
- דה-מורגן: `¬(α ∧ β) ≡ ¬α ∨ ¬β` ו-`¬(α ∧ β) ≡ ¬α ∨ ¬β`
- דיסטריביוטיביות: `α ∧ (β ∨ γ) ≡ (α ∧ β) ∨ (α ∧ γ)`
- Contrapositive: `p -> q ≡ ¬q -> ¬p`

**טאוטולוגיה:** פסוק שבטבלת האמת שלו מופיעים ערכי T בלבד.
- T בלבד = טאוטולוגיה
- F בלבד = סתירה
- שילוב = ספיק (contingent)

### טיפים מהמבחנים

**שאלת "מצא פסוקים לא שקולים" (חוזרת כל שנה!):**
- השיטה: המר הכל לצורת ¬,∧,∨ באמצעות `p -> q ≡ ¬p ∨ q`
- פשט עם דה-מורגן ודיסטריביוטיביות
- השווה - אם שונים, הם לא שקולים
- דוגמה מ-2024A: נתון `(p → (q → r)) ∧ (s ∨ ¬s)` - צריך למצוא מי לא שקול מבין 4 אפשרויות

**שאלת "כתוב פסוק שקול עם קשרים ¬,∧ בלבד":**
- דוגמה 2022A: `q * p ≡ ¬(q → p) ≡ q ∧ ¬p`
- המפתח: תמיד להמיר → ל-¬ ו-∨ ואז להשתמש בדה-מורגן

**שאלת "קבוצה אדקווטית" (adequate set):**
- {¬, ∧}, {¬, ∨}, {¬, →} כולם אדקווטיים
- NAND (↑) ו-NOR (↓) כל אחד לבדו אדקווטי
- {∧, ∨} **לא** אדקווטי (לא יכול לבטא שלילה)

**כשמבקשים "פסוק γ כך ש-α∨γ ו-β∨γ טאוטולוגיות אבל γ לא":**
- חפש את מה שחסר ל-α ול-β כדי להיות טאוטולוגיות
- דוגמה 2023B: הפתרון הפשוט: `γ = p`

### דפוסים חוזרים
1. **שאלת שקילות/אי-שקילות** - בכל מבחן! השתמש בהמרה ל-∨,∧,¬
2. **כתיבה עם קשרים מוגבלים** - חוזר לעיתים
3. **CNF/DNF** - דעי את ההבדל (CNF = conjunction of disjunctions, DNF = disjunction of conjunctions)

## 2. פרדיקטים (Predicates / Quantifiers)

### כללי זהב

**שלילת כמתים (חובה לשינון!):**
- `¬∀x(α) ≡ ∃x(¬α)`
- `¬∃x(α) ≡ ∀x(¬α)`
- שלילת כמתים מקוננים: פשוט הפוך כל כמת ושלל את הפסוק הפנימי

**פרשנות (Interpretation):**
- פרשנות = (עולם, הצבה לפרדיקטים ולקבועים)
- כדי להוכיח שפסוק **לא** טאוטולוגיה: מספיק פרשנות אחת שמחזירה F
- כדי להוכיח שפסוק **הוא** טאוטולוגיה: צריך להוכיח לכל פרשנות

### טיפים מהמבחנים

**הפרכה (Disprove) לעומת הוכחה (Prove):**
- הפרכה: תן דוגמה נגדית (פרשנות ספציפית)
- הוכחה: הוכח באופן כללי, לכל פרשנות

**שימו לב ל:**
- ∀x∃y ≠ ∃y∀x (סדר הכמתים חשוב!)
- משתנים חופשיים vs קשורים
- תחום הכמת (scope)

### דפוסים חוזרים
1. **שלילת טענה עם כמתים** - חוזר בשאלות פתוחות
2. **הוכח/הפרך** - שאלה קלאסית; להפרכה מספיק דוגמה נגדית

## 3. קבוצות (Sets)

### כללי זהב

**הגדרות בסיסיות:**
- `A∆B = (A\B) ∪ (B\A)` (הפרש סימטרי)
- `A × B = {(a,b) | a ∈ A and b ∈ B}`
- `P(A) = {S | S ⊆ A}`, ו-`|P(A)| = 2^|A|`
- `|A × B| = |A| · |B|`

**דה-מורגן לקבוצות:**
- `complement(B ∪ C) = B̄ ∩ C̄`
- `complement(B ∩ C) = B̄ ∪ C̄`
- `A\(B ∪ C) = (A\B) ∩ (A\C)`
- `A\(B ∩ C) = (A\B) ∪ (A\C)`

**דיסטריביוטיביות:**
- `A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)`
- `A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)`

### טיפים מהמבחנים

**הפרכה עם קבוצות (שאלה סופר-חוזרת!):**
- הדוגמה הנגדית הכי שימושית: `A={1}, B={1,2}, C={3}` או `A={1,2}, B={1}, C={2}`
- מופיעה כמעט בכל מבחן! (2022A Q13, 2024A Q2, 2024B Q2)
- המפתח: תבדקו אם A ⊈ B∩C, ואז תבדקו מה קורה עם ההפרש

**הוכחת שוויון/הכלה בין קבוצות:**
- להכלה A ⊆ B: יהי x ∈ A, הוכח ש-x ∈ B
- לשוויון: הוכח הכלה בשני הכיוונים
- תמיד התחילו עם "יהי x ∈ ..."

**קבוצת חזקה - טריקים:**
- `∅ ∈ P(A)` תמיד! (כי ∅ ⊆ A)
- `A ∈ P(A)` תמיד! (כי A ⊆ A)
- `P(A\B) ≠ P(A)\P(B)` (שאלה מ-2023B: ∅ ∈ P(A\B) אבל ∅ ∉ P(A)\P(B))

**הוכחה עם מכפלה קרטזית (חוזר!):**
- דוגמה 2024A Q10: `(A×A) ∪ (B×B) = (C×C)` → הוכח A∪B ⊆ C
- הטריק: קח x ∈ A∪B, אז (x,x) ∈ (A×A) ∪ (B×B) = (C×C), ולכן x ∈ C

### דפוסים חוזרים
1. **הפרכה עם counterexample** - כמעט בכל מבחן!
2. **הוכחת הכלה/שוויון** - שאלה פתוחה קלאסית
3. **P(A) ושאלות על קבוצת חזקה** - חוזר הרבה
4. **∅ כמלכודת** - שימו לב ש-∅ תמיד בקבוצת החזקה

## 4. פונקציות (Functions)

### כללי זהב

**חד-חד-ערכית (Injective / 1-1):**
- `f(x₁) = f(x₂) → x₁ = x₂`
- או שקול: `x₁ ≠ x₂ → f(x₁) ≠ f(x₂)`
- אם |A| > |B| אז אין פונקציה חח"ע מ-A ל-B

**על (Surjective / Onto):**
- לכל y ∈ B קיים x ∈ A כך ש-f(x) = y
- אם |A| < |B| אז אין פונקציה על מ-A ל-B

**הפיכה (Bijective):**
- חח"ע + על = הפיכה
- קיים f⁻¹ אם"ם f הפיכה

**הרכבת פונקציות:**
- `(g ∘ f)(x) = g(f(x))`
- g ∘ f חח"ע → f חח"ע (אבל לא בהכרח g!)
- g ∘ f על → g על (אבל לא בהכרח f!)
- f חח"ע ∧ g חח"ע → g ∘ f חח"ע
- f על ∧ g על → g ∘ f על

**הפיכות שמאלית/ימנית:**
- הפיכה שמאלית (left inverse): קיים g כך ש-g ∘ f = id → f חח"ע
- הפיכה ימנית (right inverse): קיים g כך ש-f ∘ g = id → f על

### טיפים מהמבחנים

**שאלת "האם קיימות קבוצות ופונקציות כך ש..." (חוזרת!):**
- דוגמה 2024A Q3: f:A→B חח"ע, g∘f על אבל לא חח"ע? כן!
  - `A={1,2}, B={3,4,5}, C={6}, f={(1,3),(2,4)}, g={(3,6),(4,6),(5,6)}`
- המפתח: שחקו עם גדלי הקבוצות

**שאלת "מצא הפיכה ימנית/שמאלית":**
- 2022A Q3: `h(f,n,m) = (f(n),f(m))`, מצא הפיכה ימנית
  - הפיכה ימנית: `g(n,m) = (i_N, n, m)` (השתמשו בפונקצית הזהות)

**הוכחת חח"ע/על דרך ביטוי אלגברי:**
- 2024B Q10: f(n) = 2g(n)-1 → f לא על (0 לא בתמונה), ואם f חח"ע אז g חח"ע
- הטריק: הוכחה בשלילה - אם יש n כך ש-f(n)=0 אז g(n)=1/2 ∉ ℕ

**f: P(A)×P(B) → P(C) ופונקציות על קבוצות חזקה:**
- 2023B: `f(S,T) = S ∪ T` כשידוע A∩B=∅, A∪B=C → f הפיכה
  - `f⁻¹(K) = (K∩A, K∩B)`

### דפוסים חוזרים
1. **הוכחה ש-f (לא) חח"ע / (לא) על** - כל מבחן
2. **הרכבת פונקציות** - שאלה קלאסית על שימור חח"ע/על
3. **מציאת הפיכה / הפיכה חלקית** - חוזר הרבה
4. **פונקציות מ/ל-P(A)** - חוזר

## 5. קומבינטוריקה (Combinatorics)

### כללי זהב

**כלי הספירה:**
- סדרה עם חזרות (n אפשרויות, k מקומות): `n^k`
- סדרה ללא חזרות (תמורה חלקית): `n!/(n-k)!`
- תת-קבוצה (בחירה ללא סדר): `C(n,k) = n!/(k!(n-k)!)`
- מולטינום: `C(n; k₁,k₂,...,kₘ) = n!/(k₁!·k₂!·...·kₘ!)` כש-Σkᵢ = n

**כוכבים ומחיצות (Stars and Bars):**
- חלוקת n כדורים זהים ל-k תאים: `C(n+k-1, k-1)` = `C(n+k-1, n)`
- עם אילוץ xᵢ ≥ aᵢ: שנה משתנה yᵢ = xᵢ - aᵢ

**הבינום של ניוטון:**
- `(x+y)^n = Σ C(n,k) x^k y^(n-k)`
- מולטינום: `(x₁+x₂+...+xₘ)^n = Σ C(n;k₁,...,kₘ) x₁^k₁ · ... · xₘ^kₘ`

### טיפים מהמבחנים

**מקדם בפיתוח (שאלה חוזרת כל שנה!):**

**חלוקת כדורים לתאים עם אילוצים:**

  - תשובה: `C(8,2)·C(6,2)·C(4,2)·C(2,2) = 8!/(2!⁴)` = מולטינום!

## 6. הכלה והדחה (Inclusion-Exclusion)

### כללי זהב

**הנוסחה:**
- `|A₁ ∪ A₂ ∪ ... ∪ Aₙ| = Σ|Aᵢ| - Σ|Aᵢ∩Aⱼ| + Σ|Aᵢ∩Aⱼ∩Aₖ| - ... + (-1)ⁿ⁺¹|A₁∩...∩Aₙ|`
- לספירת מה שלא באף קבוצה: `|Ū| = |U| - |A₁ ∪ ... ∪ Aₙ|`

**שימושים קלאסיים:**
- ספירת פונקציות על (surjections): `Σ(-1)ⁱ C(n,i)(n-i)^k` (מ-k ל-n)
- שיבושים (Derangements): `Dₙ = n! Σ(-1)ⁱ/i!` (i מ-0 עד n)
- פונקצית אוילר: `φ(n) = n · Π(1 - 1/pᵢ)` (pᵢ הגורמים הראשוניים של n)

### טיפים מהמבחנים

**שאלת "בחירה עם לפחות אחד מכל סוג" (חוזרת!):**
- 2024A Q4: 6 מתוך 15 קורסים (6 מתמטיקה, 6 מדמ"ח, 3 פיזיקה), לפחות 1 מכל תחום
  - Aᵢ = בחירות בלי תחום i
  - `C(15,6) - C(9,6) - C(9,6) - C(12,6) + C(6,6) + C(6,6) + C(3,6)`
  - שימו לב: C(3,6) = 0 כי 3 < 6

**שאלת "פתרונות טבעיים למשוואה עם אילוץ עליון" (סופר-חוזרת!):**
- 2024B Q6: `x₁+...+x₆ = 15` עם `0 ≤ xᵢ ≤ 5`
  - שים yᵢ = 5-xᵢ, עכשיו: `y₁+...+y₆ = 15` עם `0 ≤ yᵢ ≤ 5`
  - הכלה והדחה: `C(20,5) - C(6,1)C(14,5) + C(6,2)C(8,5) - ...`
- 2023B Q7: `x₁+...+x₆ = 50` עם `5 ≤ xᵢ ≤ 10`

## 7. יחסים (Relations)

### כללי זהב

**תכונות יחס:**
- **רפלקסיבי:** ∀a ∈ A: aRa
- **סימטרי:** ∀a,b: aRb → bRa
- **אנטי-סימטרי:** ∀a,b: (aRb ∧ bRa) → a=b
- **טרנזיטיבי:** ∀a,b,c: (aRb ∧ bRc) → aRc
- **א-רפלקסיבי (irreflexive):** ∀a: ¬(aRa)

**סוגי יחסים:**
- **יחס שקילות:** רפלקסיבי + סימטרי + טרנזיטיבי
  - מחלק את הקבוצה למחלקות שקילות
  - |A/R| = מספר מחלקות השקילות
- **סדר חלקי חלש (weak partial order):** רפלקסיבי + אנטי-סימטרי + טרנזיטיבי
- **סדר חלקי חזק (strict partial order):** א-רפלקסיבי + אנטי-סימטרי + טרנזיטיבי
- **סדר מלא (total order):** סדר חלקי + ∀a,b: (aRb ∨ bRa)

**דיאגרמת האסה (Hasse Diagram):**
- **מינימלי:** אין אף איבר קטן ממנו (יכולים להיות כמה)
- **מקסימלי:** אין אף איבר גדול ממנו (יכולים להיות כמה)
- **מינימום:** מינימלי יחיד (אם קיים)
- **מקסימום:** מקסימלי יחיד (אם קיים)

### טיפים מהמבחנים

**הוכח/הפרך שיחס הוא שקילות/סדר (סופר-חוזר!):**
- 2024A Q11: xQy ⟺ ∀z(xRz → yRz)
  - Q רפלקסיבי? כן (טריוויאלי - אם xRz אז xRz)
  - Q סימטרי? לא! תן דוגמה נגדית עם R ספציפי
- 2024B Q11: (a,b)R(c,d) iff a·d = b·c
  - R שקילות? כן! (רפלקסיבי, סימטרי, טרנזיטיבי - כי a/b = c/d)
  - (a,b)S(c,d) iff a·c = b·d - לא שקילות ולא סדר (לא רפלקסיבי: 1·1 ≠ 2·2)

**דוגמה נגדית ליחס (חוזר מאוד!):**
- 2024B Q8: יחס שהוא לא סימטרי, לא אנטי-סימטרי, אבל פונקציה הפיכה שמאלית
  - `A={1,2,3,4,5}, R={(1,2),(2,1),(3,4),(4,5),(5,3)}`

**מחלקות שקילות (חוזר!):**
