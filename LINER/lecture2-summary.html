1. משפט 1 — קיום ויחידות של העתקה לינארית

משפט 1 (קיום ויחידות)
יהי \(\mathbb{F}\) שדה ויהיו \(V,W\) מרחבים וקטוריים מעל \(\mathbb{F}\).
נניח ש-\(V\) נוצר סופית ויהיו \(v_1, \ldots, v_n\) בסיס של \(V\).
יהיו \(w_1, \ldots, w_n \in W\) (סתם וקטורים, לאו דווקא בסיס).

אזי קיימת ה"ל יחידה \(T: V \to W\) כך ש-\(T(v_i) = w_i\) לכל \(1 \leq i \leq n\).
המשמעות
  • \(w_1, \ldots, w_n\) לא בהכרח בסיס של \(W\).
  • המימד של \(W\) אינו בהכרח \(n\).
  • מספיק להגדיר ה"ל על הבסיס — היא נקבעת באופן יחיד על כל המרחב.
הוכחת קיום
יהי \(v \in V\). מהיות \(v_1, \ldots, v_n\) בסיס, קיימים \(\alpha_1, \ldots, \alpha_n \in \mathbb{F}\) יחידים כך ש-\(v = \alpha_1 v_1 + \cdots + \alpha_n v_n\).
נגדיר \(T: V \to W\) באופן הבא:
\(\boxed{T(v) = \alpha_1 w_1 + \cdots + \alpha_n w_n}\)
\(T\) מוגדרת היטב מכיוון ש-\(\alpha_1, \ldots, \alpha_n\) יחידים.
בנוסף, מהיות \(W\) תמ"ו (סגור לצ"ל) נקבל ש-\(\alpha_1 w_1 + \cdots + \alpha_n w_n \in W\).

צריך להוכיח שני דברים:

1. \(T(v_i) = w_i\): יהי \(1 \leq i \leq n\). אזי:
\(T(v_i) = T(0_\mathbb{F} v_1 + \cdots + 1_\mathbb{F} v_i + \cdots + 0_\mathbb{F} v_n) = 0_\mathbb{F} w_1 + \cdots + 1_\mathbb{F} w_i + \cdots + 0_\mathbb{F} w_n = w_i \;\; \checkmark\)
2. \(T\) ה"ל: יהיו \(u, v \in V\) ויהיו \(\alpha, \beta \in \mathbb{F}\).
נכתוב \(u = \alpha_1 v_1 + \cdots + \alpha_n v_n\) ו-\(v = \beta_1 v_1 + \cdots + \beta_n v_n\). אזי:
\(T(\alpha u + \beta v) = T\bigl((\alpha\alpha_1 + \beta\beta_1)v_1 + \cdots + (\alpha\alpha_n + \beta\beta_n)v_n\bigr)\)
\(= (\alpha\alpha_1 + \beta\beta_1)w_1 + \cdots + (\alpha\alpha_n + \beta\beta_n)w_n\)
\(= \alpha(\alpha_1 w_1 + \cdots + \alpha_n w_n) + \beta(\beta_1 w_1 + \cdots + \beta_n w_n) = \alpha T(u) + \beta T(v) \;\; \checkmark\)
הוכחת יחידות
תהי \(S: V \to W\) ה"ל המקיימת \(S(v_i) = w_i\) לכל \(1 \leq i \leq n\). נוכיח \(S = T\).

יהי \(v \in V\). לכן קיימים \(\alpha_1, \ldots, \alpha_n\) כך ש-\(v = \alpha_1 v_1 + \cdots + \alpha_n v_n\).
\(S(v) = S(\alpha_1 v_1 + \cdots + \alpha_n v_n) = \alpha_1 S(v_1) + \cdots + \alpha_n S(v_n) = \alpha_1 w_1 + \cdots + \alpha_n w_n = T(v)\)
לכל \(v \in V\) מתקיים \(S(v) = T(v)\), לכן \(S = T\). \(\blacksquare\)

2. תרגיל 1 — מציאת ה"ל לפי ערכים על בסיס

תרגיל 1
תהי \(T: \mathbb{R}^2 \to \mathbb{R}^2\) ה"ל המקיימת: \(T\binom{1}{1} = \binom{2}{1}\), \(\;T\binom{1}{-1} = \binom{1}{-1}\).
א. מצאו את \(T\binom{x}{y}\) לכל \(\binom{x}{y} \in \mathbb{R}^2\).
ב. הוכיחו ש-\(T\) הפיכה וחשבו את \(T^{-1}\binom{1}{0}\).
פתרון א
יהי \(\binom{x}{y} \in \mathbb{R}^2\). נסמן: \(T\binom{x}{y} = \binom{ax+by}{cx+dy}\).
מהנתון השמאלי (\(x=1, y=1\)):
\(a + b = 2, \quad c + d = 1\)
מהנתון הימני (\(x=1, y=-1\)):
\(a - b = 1, \quad c - d = -1\)
נפתור: \(a = 1.5,\; b = 0.5,\; c = 0,\; d = 1\). לכן:
\(\boxed{T\binom{x}{y} = \binom{1.5x + 0.5y}{y}}\)
פתרון ב
נשים לב ש-\(\left\{\binom{2}{1}, \binom{1}{-1}\right\}\) וגם \(\left\{\binom{1}{1}, \binom{1}{-1}\right\}\) הם בסיסים ל-\(\mathbb{R}^2\).
לכן \(T\) מעתיקה בסיס לבסיס — ולפי משפט 13 מהרצאה 1, היא הפיכה.

חישוב \(T^{-1}\binom{1}{0}\): נבטא \(\binom{1}{0}\) כצ"ל של איברי הבסיס:
\(\binom{1}{0} = \frac{1}{3}\binom{2}{1} + \frac{1}{3}\binom{1}{-1}\)
לכן:
\(T^{-1}\binom{1}{0} = \frac{1}{3} T^{-1}\binom{2}{1} + \frac{1}{3} T^{-1}\binom{1}{-1} = \frac{1}{3}\binom{1}{1} + \frac{1}{3}\binom{1}{-1} = \binom{2/3}{0}\)

3. משפט 2 — משפט המימדים השני (Rank-Nullity Theorem)

משפט 2 — משפט המימדים השני
יהי \(\mathbb{F}\) שדה ויהיו \(V,W\) מ"ו מעל \(\mathbb{F}\). נניח ש-\(V\) נוצר סופית.
תהי \(T: V \to W\) ה"ל. אזי:
\(\boxed{\dim V = \dim Ker\,T + \dim Im\,T}\)
הוכחה (מלאה)
  • מהיות \(V\) נוצר סופית, קיים \(0 \leq n \in \mathbb{N}\) כך ש-\(\dim V = n\).
  • מכיוון ש-\(Ker\,T \subseteq V\) תמ"ו, הרי שקיים \(0 \leq k \leq n\) כך ש-\(\dim Ker\,T = k\).
  • נוכיח ש-\(\dim Im\,T = n - k\) ובזאת נסיים.

מכיוון שמימד \(Ker(T)\) הוא \(k\), קיימים \(v_1, \ldots, v_k \in Ker\,T\) אשר מהווים בסיס לגרעין.

מכיוון שהם בת"ל, לפי משפט (מלינארית 1), הם ניתנים להשלמה לבסיס של \(V\).
כלומר קיימים \(v_{k+1}, \ldots, v_n \in V\) כך ש-\(v_1, \ldots, v_n\) מהווים בסיס ל-\(V\).

נתבונן בתמונות \(w_{k+1} = T(v_{k+1}), \ldots, w_n = T(v_n) \in W\).

נוכיח ש-\(w_{k+1}, \ldots, w_n\) בסיס של \(Im\,T\) ונסיק ש-\(\dim Im\,T = n - k\).

פורשים:

מהמשפט "פורשת לפורשת" (הרצ' 1 משפט 11) נובע ש:

\(Im\,T = span\{T(v_1), \ldots, T(v_k), T(v_{k+1}), \ldots, T(v_n)\}\)

אבל \(T(v_i) = 0_W\) לכל \(1 \leq i \leq k\) (כי \(v_i \in Ker\,T\)). לכן:

\(Im\,T = span\{0_W, w_{k+1}, \ldots, w_n\} = span\{w_{k+1}, \ldots, w_n\} \;\; \checkmark\)

בת"ל:

יהיו \(\alpha_{k+1}, \ldots, \alpha_n \in \mathbb{F}\) המקיימים:

\(\alpha_{k+1} w_{k+1} + \cdots + \alpha_n w_n = 0_W\)
\(\alpha_{k+1} T(v_{k+1}) + \cdots + \alpha_n T(v_n) = 0_W\)

\(T\) ה"ל, לכן:

\(T(\alpha_{k+1} v_{k+1} + \cdots + \alpha_n v_n) = 0_W\)

לכן \(\alpha_{k+1} v_{k+1} + \cdots + \alpha_n v_n \in Ker\,T\).

מהיות \(Ker\,T = span\{v_1, \ldots, v_k\}\), קיימים \(\beta_1, \ldots, \beta_k \in \mathbb{F}\) כך ש:

\(\alpha_{k+1} v_{k+1} + \cdots + \alpha_n v_n = \beta_1 v_1 + \cdots + \beta_k v_k\)
\(-\beta_1 v_1 - \cdots - \beta_k v_k + \alpha_{k+1} v_{k+1} + \cdots + \alpha_n v_n = 0_V\)

אבל \(v_1, \ldots, v_n\) בסיס ל-\(V\) ובפרט בת"ל, לכן:

\(\beta_1 = \cdots = \beta_k = \alpha_{k+1} = \cdots = \alpha_n = 0_\mathbb{F} \;\; \checkmark\)
\(\blacksquare\)

4. מסקנות ממשפט המימדים

תזכורת
\(\dim V = \dim Ker\,T + \dim Im\,T\)
מסקנה — קשר בין ממדים לחח"ע ועל
יהי \(\mathbb{F}\) שדה, \(V,W\) נוצרים סופית, \(T \in Hom(V,W)\). אזי:
תנאימסקנה
\(\dim V = \dim W\)\(T\) חח"ע אם"ם \(T\) על
\(\dim V < \dim W\)\(T\) אינה על (בוודאות)
\(\dim V > \dim W\)\(T\) אינה חח"ע (בוודאות)
הוכחה (מסקנה 1: \(\dim V = \dim W\))
\(T \text{ חח"ע} \iff Ker\,T = \{0_V\} \iff \dim Ker\,T = 0\)
\(\iff \dim Im\,T = \dim V = \dim W \iff Im\,T = W \iff T \text{ על}\)
\(\blacksquare\)
הוכחה (מסקנה 2: \(\dim V < \dim W \Rightarrow T\) אינה על)
נניח בשלילה ש-\(T\) על. לכן \(Im\,T = W\) ו-\(\dim Im\,T = \dim W\).
לפי משפט המימדים: \(\dim V = \dim Ker\,T + \dim W\).
נציב: \(\dim V = \dim Ker\,T + \dim W \geq \dim W > \dim V\). סתירה! \(\blacksquare\)
הוכחה (מסקנה 3: \(\dim V > \dim W \Rightarrow T\) אינה חח"ע)
נניח בשלילה ש-\(T\) חח"ע. לכן \(Ker\,T = \{0_V\}\) ו-\(\dim Ker\,T = 0\).
לפי משפט המימדים: \(\dim V = 0 + \dim Im\,T\).
לכן \(\dim V = \dim Im\,T \leq \dim W\). בסתירה לכך ש-\(\dim V > \dim W\). \(\blacksquare\)

5. תרגיל 2 — הפרכה עם משפט המימדים

תרגיל 2
הוכיחו או הפריכו: קיימת \(T \in Hom(\mathbb{R}^3, \mathbb{R}^3)\) שעבורה \(Ker\,T = Im\,T\).
הפרכה
נניח בשלילה שקיימת העתקה כנ"ל.
לפי משפט המימדים:
\(3 = \dim \mathbb{R}^3 = \dim Ker\,T + \dim Im\,T = 2 \cdot \dim Im\,T\)
נקבל \(\dim Im\,T = 1.5\) — סתירה! (מימד חייב להיות מספר שלם) \(\blacksquare\)

6. תרגיל 3 — הוכחה עם משפט המימדים (\(\mathbb{R}^4\))

תרגיל 3
הוכיחו או הפריכו: קיימת \(T \in Hom(\mathbb{R}^4, \mathbb{R}^4)\) שעבורה \(Ker\,T = Im\,T\).
הוכחה (בנייה!)
ממשפט הקיום והיחידות, קיימת ה"ל יחידה \(T: \mathbb{R}^4 \to \mathbb{R}^4\) כך ש:
\(T(\vec{e}_1) = \vec{0}, \quad T(\vec{e}_2) = \vec{0}, \quad T(\vec{e}_3) = \vec{e}_1, \quad T(\vec{e}_4) = \vec{e}_2\)
נחשב את התמונה:
\(Im(T) = span\{T(\vec{e}_1), T(\vec{e}_2), T(\vec{e}_3), T(\vec{e}_4)\} = span\{\vec{0}, \vec{0}, \vec{e}_1, \vec{e}_2\} = span\{\vec{e}_1, \vec{e}_2\}\)
מכיוון ש-\(T(\vec{e}_1) = \vec{0}\) ו-\(T(\vec{e}_2) = \vec{0}\), הרי ש-\(\vec{e}_1, \vec{e}_2 \in Ker(T)\).
ולכן \(span\{\vec{e}_1, \vec{e}_2\} \subseteq Ker(T)\).

לפי משפט המימדים: \(4 = \dim Ker(T) + \dim Im(T)\).
ידוע ש-\(\dim Im(T) = 2\), לכן \(\dim Ker(T) = 2\).
מכיוון ש-\(span\{\vec{e}_1, \vec{e}_2\} \subseteq Ker(T)\) והמימדים שווים:
\(Ker(T) = span\{\vec{e}_1, \vec{e}_2\} = Im(T) \;\; \checkmark\)
\(\blacksquare\)

7. תרגיל 4 — נגזרת כהעתקה לינארית

תרגיל 4
תהי \(T: \mathbb{R}_3[x] \to \mathbb{R}_2[x]\) ההעתקה המוגדרת כך: \(T(p(x)) = p'(x)\) (נגזרת).
מצאו בסיס ל-\(Ker(T)\) ול-\(Im(T)\).
פתרון
הגרעין:
\(Ker(T) = \{p(x) \in \mathbb{R}_3[x] \;|\; p'(x) = 0\} = \{p(x) \;|\; p(x) = c\} = \mathbb{R}_0[x] = span\{1\}\)
כלומר בסיס ל-\(Ker(T)\) הוא \(\{p_0(x) = 1\}\), ומימד 1.

לפי משפט המימדים:
\(\dim \mathbb{R}_3[x] = \dim Ker(T) + \dim Im(T)\)
\(4 = 1 + \dim Im(T) \;\Rightarrow\; \dim Im(T) = 3\)
נשים לב שגם \(\dim \mathbb{R}_2[x] = 3\). כמו כן \(Im(T) \subseteq \mathbb{R}_2[x]\).
לכן לפי משפט (מימדים שווים): \(Im(T) = \mathbb{R}_2[x]\).
כלומר בסיס ל-\(Im(T)\) הוא \(\{1, x, x^2\}\).

8. מרחב השורות, העמודות והאפס של מטריצה

הגדרות (תזכורת מלינארית 1)
יהי \(\mathbb{F}\) שדה ותהי \(A \in M_{m \times n}(\mathbb{F})\). תהיינה \(\vec{C}_1, \ldots, \vec{C}_n \in \mathbb{F}^m\) עמודות \(A\) ותהיינה \(\vec{R}_1, \ldots, \vec{R}_m \in \mathbb{F}^n\) שורות \(A\). נגדיר:

1. מרחב השורות: \(\;row(A) = span\{\vec{R}_1, \ldots, \vec{R}_m\}\)
2. מרחב העמודות: \(\;col(A) = span\{\vec{C}_1, \ldots, \vec{C}_n\}\)
3. מרחב האפס: \(\;null(A) = \{\vec{x} \in \mathbb{F}^n \;|\; A\vec{x} = \vec{0}\}\)
דרגת שורות ודרגת עמודות
4. דרגת השורות: \(\;row\text{-}rank(A) = \dim row(A)\)
5. דרגת העמודות: \(\;col\text{-}rank(A) = \dim col(A)\)

ידוע (הוכח בלינארית 1): \(row\text{-}rank(A) = rank\,A\). דרגת השורות = דרגת המטריצה.
הערה 1 — \(col(A) = Im(T_A)\)
ידוע שלכל \(\vec{x} \in \mathbb{F}^n\):
\(A\vec{x} = x_1\vec{C}_1 + x_2\vec{C}_2 + \cdots + x_n\vec{C}_n\)
לכן:
\(col(A) = span\{\vec{C}_1, \ldots, \vec{C}_n\} = \{A\vec{x} \;|\; \vec{x} \in \mathbb{F}^n\} = \{T_A(\vec{x}) \;|\; \vec{x} \in \mathbb{F}^n\} = \boxed{Im(T_A)}\)
הערה 2 — \(null(A) = Ker(T_A)\)
\(null(A) = \{\vec{x} \in \mathbb{F}^n \;|\; A\vec{x} = \vec{0}\} = \boxed{Ker(T_A)}\)
לכן:
\(\dim null(A) = \dim Ker(T_A), \qquad \dim col(A) = \dim Im(T_A)\)

9. מסקנה 4 — Rank-Nullity למטריצות

מסקנה 4 — Rank-Nullity Theorem
תהי \(A \in M_{m \times n}(\mathbb{F})\). אזי:
\(\boxed{\dim null(A) + rank(A) = n}\)
כלומר: מספר המשתנים החופשיים + מספר המובילים = מספר המשתנים.
הוכחה
נתבונן בה"ל \(T_A: \mathbb{F}^n \to \mathbb{F}^m\) המוגדרת \(T_A(\vec{x}) = A\vec{x}\).
ממשפט המימדים השני:
\(\dim \mathbb{F}^n = \dim Ker(T_A) + \dim Im(T_A)\)
\(n = \dim null(A) + \dim col(A)\)
נותר להראות ש-\(\dim col(A) = rank(A)\).
ידוע ש-\(rank(A) = row\text{-}rank(A)\). תהי \(C\) הצורה הקנונית של \(A\). ידוע ש-\(row\text{-}rank(A) = row - rank(C)\) (שהינו מספר האיברים המובילים ב-\(C\)).
מכיוון שכל איבר מוביל נמצא גם בעמודה אחרת: \(\dim col(A) = rank(A)\). \(\blacksquare\)
שימו לב
הדירוג שומר על מרחב השורות ולא בהכרח על מרחב העמודות, אבל הוא כן שומר על המימד של מרחב העמודות!

10. איזומורפיזם

הגדרה: איזומורפיזם
יהי \(\mathbb{F}\) שדה ויהיו \(V,W\) מרחבים וקטוריים מעל \(\mathbb{F}\). תהי \(T: V \to W\).
נאמר ש-\(T\) היא איזומורפיזם אם:
  1. \(T\) היא העתקה לינארית.
  2. \(T\) הפיכה (חח"ע ועל).
הגדרה: אוטומורפיזם
אם \(V = W\) נאמר ש-\(T\) היא אוטומורפיזם.
הגדרה: מרחבים איזומורפיים
נאמר ש-\(V,W\) הם איזומורפיים ונסמן \(V \cong W\) אם קיימת \(T: V \to W\) שהיא איזומורפיזם.

iso = שווה, morph = צורה. מרחבים איזומורפיים הם למעשה אותו מרחב בשינוי שמות.
הערה 3 — איזומורפיזם הוא יחס שקילות
תכונהטענהנימוק
רפלקסיבי\(V \cong V\)\(id_V\) היא איזומורפיזם
סימטריאם \(V \cong W\) אז \(W \cong V\)\(T^{-1}\) ה"ל הפיכה (הרצ' 1 משפט 9)
טרנזיטיביאם \(V \cong W\) ו-\(W \cong U\) אז \(V \cong U\)\(S \circ T\) ה"ל (הרצ' 1 משפט 8) והרכבת הפיכות הפיכה
