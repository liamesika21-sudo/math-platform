## 1. שדות (Fields) — הרצאות 1-4

### מה זה שדה?
שדה זה בעצם קבוצה F עם שתי פעולות — חיבור (+) וכפל (·) — שמתנהגות "יפה".
תחשבו על זה ככה: המספרים הרגילים שאנחנו מכירים (Q, R) הם שדות, כי אפשר לחבר, לכפול, לחלק (חוץ מב-0), והכל עובד כמו שצריך.

### האקסיומות (הכללים) של שדה:

**חיבור:**
- **סגירות לחיבור**: אם a,b בשדה, אז גם a+b בשדה
- **חילופיות**: a+b = b+a (לא משנה הסדר)
- **קיבוציות**: (a+b)+c = a+(b+c) (לא משנה איפה שמים סוגריים)
- **איבר אדיש (אפס)**: קיים 0_F כך ש a+0 = a (חיבור אפס לא משנה כלום)
- **נגדי**: לכל a יש מינוס a, כך ש a+(-a) = 0

**כפל:**
- **סגירות לכפל**: אם a,b בשדה, אז גם a·b בשדה
- **חילופיות (קומוטטיביות) כפל**: a·b = b·a
- **קיבוציות (אסוציאטיביות) כפל**: a·(b·c) = (a·b)·c
- **איבר יחידה**: קיים 1_F כך ש a·1 = a (כפל ב-1 לא משנה כלום)
- **הופכי**: לכל a≠0 יש a⁻¹ כך ש a·a⁻¹ = 1

**קשר בין חיבור לכפל:**
- **דיסטריביוטיביות (פילוג)**: a·(b+c) = a·b + a·c

**חשוב!** 0_F ≠ 1_F (האפס והיחידה הם איברים שונים)

### משפטים חשובים על שדות:

- **Q ו-R הם שדות** — עם הפעולות הרגילות של חיבור וכפל
- **יחידות האפס**: האפס של שדה הוא יחיד (יש רק אחד כזה)
- **חוק הצמצום בחיבור**: אם x+z = y+z אז x=y (אפשר "לקצר" חיבור)
- **יחידות ההופכי לחיבור**: לכל x יש נגדי יחיד, שזה -x, ומתקיים x+y=0 אמ"מ y=-x
- **כפל באפס**: x·0 = 0 (תמיד! לא משנה מה x)
- **אין מחלקי אפס**: אם x·y = 0 אז בהכרח x=0 או y=0 (זה סופר חשוב!)
- **יחידות ה-1**: היחידה 1_F היא יחידה
- **יחידות ההופכי לכפל**: לכל x≠0 יש הופכי יחיד y=x⁻¹
- **חוק הצמצום בכפל**: אם z≠0 ו x·z = y·z אז x=y
- **תכונות הנגדי**: -0=0, -(-x)=x, -1·x = -x, (-x)·(-y) = x·y, -(x·y) = (-x)·y = x·(-y)
- **תכונות ההופכי**: 1⁻¹=1, (x⁻¹)⁻¹=x, (x·y)⁻¹ = y⁻¹·x⁻¹

### חלוקה (mod) ומספרים שלמים:

**חלוקה עם שארית**: לכל a∈Z ו-b∈N, קיימים q,r יחידים כך ש a = q·b + r כאשר 0 ≤ r < b.
הסימון: a mod b = r

**מחלק**: a מחלק את b (a|b) אם קיים k∈Z כך ש b = k·a

**שקילות מודולו n**: a ≡ b (mod n) אם n | (a-b), כלומר ההפרש מתחלק ב-n

**Zₙ = {0, 1, 2, ..., n-1}** — קבוצת השאריות מודולו n
- חיבור: a ⊕ b = (a+b) mod n
- כפל: a ⊙ b = (a·b) mod n

### מספרים ראשוניים וזרים:
- **ראשוני (prime)**: p≥2 שהמחלקים היחידים שלו הם 1 ו-p
- **זרים (coprime)**: a,b כך שהמחלק המשותף היחיד הוא 1 או -1
- **משפט**: אם n לא ראשוני → Zₙ לא שדה
- **משפט**: אם a∈Zₙ ו-a זר ל-n → יש ל-a הופכי ב-Zₙ
- **מסקנה**: n ראשוני ⟹ Zₙ שדה!

### משפט בזו (Bézout):
אם a,b∈Z זרים, אז קיימים x,y∈Z כך ש ax + by = 1

### תת שדה:
F₁ ⊆ F הוא תת שדה אם הוא בעצמו שדה עם אותן פעולות.
**תנאי מקוצר**: F₁ סגור לחיבור ולכפל, סגור לנגדי ולהופכי, ו-0,1 שייכים ל-F₁.

### מספרים מרוכבים C:
z = a + bi כאשר a,b ממשיים ו-i² = -1

| פעולה | נוסחה |
|-------|-------|
| חלק ממשי | Re(z) = a |
| חלק מדומה | Im(z) = b |
| שוויון | a₁+b₁i = a₂+b₂i ⟺ a₁=a₂ וגם b₁=b₂ |
| סכום | z+w = (a+c) + (b+d)i |
| מכפלה | z·w = (ac-bd) + (ad+bc)i |
| צמוד | z̄ = a - bi |
| ערך מוחלט (מודול) | |z| = √(a²+b²) |
| הופכי | z⁻¹ = z̄/|z|² |

**תכונות הצמוד**: צמוד של מכפלה = מכפלת הצמודים. מודול של מכפלה = מכפלת המודולים.

## 2. מערכות ליניאריות — הרצאות 6-4

### מה זו מערכת ליניארית?
מערכת של m משוואות עם n נעלמים (x₁,...,xₙ) מעל שדה F.
כל משוואה נראית ככה: a₁x₁ + a₂x₂ + ... + aₙxₙ = b

### מטריצת מקדמים מורחבת A⁺:
זו הטבלה שמייצגת את המערכת — כל שורה = משוואה, כל עמודה = מקדם של משתנה, ועמודה אחרונה = האגף הימני.

### מטריצת מקדמים מנותקת A:
אותו דבר בלי העמודה האחרונה (בלי ה-b-ים).

### קבוצת המטריצות M_{m×n}(F):
כל המטריצות מסדר m×n עם איברים מהשדה F. כש-n=1 מסמנים Fᵐ (וקטורי עמודה).

### פתרון פרטי:
רשימה (s₁,...,sₙ) שאם מציבים בכל המשוואות — כולן מתקיימות.

### פתרון כללי:
קבוצת כל הפתרונות הפרטיים.

### מערכות שקולות:
שתי מערכות שקולות אם יש להן אותה קבוצת פתרונות.

### פעולות שורה אלמנטריות:
שלוש פעולות שאפשר לעשות על מטריצה בלי לשנות את קבוצת הפתרונות:
1. **כפל שורה בסקלר**: Rᵢ → αRᵢ (כאשר α≠0)
2. **החלפת שורות**: Rᵢ ↔ Rⱼ
3. **הוספת כפולה**: Rᵢ → Rᵢ + αRⱼ

**חשוב!** מערכות ששקולות שורה → שקולות (אותם פתרונות!)

### מקדם מוביל:
האיבר הכי שמאלי שאינו אפס בשורה.

### מטריצה מדורגת (EF):
1. שורות אפסים (אם יש) נמצאות למטה
2. מקדם מוביל של שורה נמצא ימינה למקדם המוביל של השורה שמעליו

### מטריצה מדורגת קנונית (REF):
בנוסף למדורגת:
1. כל מקדם מוביל שווה ל-1
2. כל מקדם מוביל הוא האיבר היחיד שאינו 0 בעמודה שלו
3. היא **יחידה** — כל מטריצה שקולת שורות למדורגת קנונית אחת ויחידה!

### משתנה קשור ומשתנה חופשי:
- **קשור**: עמודה שיש בה מקדם מוביל → המשתנה "נקבע" ע"י האחרים
- **חופשי**: עמודה בלי מקדם מוביל → אפשר לבחור לו כל ערך

### איך להביא מטריצה לצורה מדורגת (אלגוריתם):
1. מוצאים את העמודה הכי שמאלית שאינה אפס
2. מחליפים שורות כדי שהאיבר שאינו אפס יהיה בראש
3. משתמשים בהוספת כפולה כדי לאפס הכל מתחת למוביל
4. מתעלמים מהשורה העליונה וחוזרים על 1-3
5. עולים חזרה ומאפסים גם מעל (לקנונית), ומנרמלים מובילים ל-1

### Rank (דרגה):

**הגדרה**: rank A = מספר האיברים המובילים בצורה הקנונית של A.

**הערות חשובות**:
- rank A = rank A⁺ אמ"מ המערכת **עקבית** (יש פתרון) — אין שורת סתירה!
- שורת סתירה: (0 0 ... 0 | b) כאשר b≠0
- 0 ≤ rank A ≤ min(m,n)
- 0 ≤ rank A⁺ ≤ n+1

### מספר פתרונות:
- **אם rank A < rank A⁺**: אין פתרון (∅)
- **אם rank A = rank A⁺ = n** (מספר המשתנים): פתרון יחיד
- **אם rank A = rank A⁺ < n**: אינסוף פתרונות
  - אם F אינסופי → אינסוף פתרונות
  - אם F סופי → |F|^(n - rank A) פתרונות

**מסקנה**: אם n < m (יותר משוואות ממשתנים) ו rank A ≤ m ≤ n → אין פתרון יחיד (אלא 0 או אינסוף).

### מערכת הומוגנית:
מערכת שכל ה-bᵢ שווים 0. תמיד יש לה פתרון — הפתרון הטריוויאלי (0,0,...,0).

## 3. מרחב וקטורי — הרצאות 7-16

### מה זה מרחב וקטורי?
קבוצה V של "וקטורים" מעל שדה F (הסקלרים) עם שתי פעולות — חיבור וקטורים וכפל בסקלר — שמקיימות 10 אקסיומות.

### 10 האקסיומות:
1. **סגירות לחיבור**: u+v ∈ V
2. **חילופיות בחיבור**: u+v = v+u
3. **קיבוציות בחיבור**: (u+v)+w = u+(v+w)
4. **אדיש לחיבור**: קיים 0_V כך ש u+0=u
5. **נגדי לחיבור**: לכל u קיים -u כך ש u+(-u) = 0
6. **סגירות לכפל בסקלר**: α·u מוגדר וגם ∈ V
7. **קיבוציות בכפל בסקלר**: α·(β·u) = (α·β)·u
8. **אדיש לכפל בסקלר**: 1·u = u
9. **פילוג וקטורים**: α(u+v) = αu + αv
10. **פילוג סקלרים**: (α+β)u = αu + βu

### דוגמאות חשובות:
- **Fⁿ** הוא מרחב וקטורי מעל F לכל n
- **M_{m×n}(F)** — קבוצת המטריצות — הוא מרחב וקטורי
- **F[x]** — קבוצת הפולינומים — הוא מרחב וקטורי

### תכונות מרחב וקטורי (נובעות מהאקסיומות):
1. האיבר האדיש לחיבור **יחיד**
2. לכל וקטור v יש נגדי **יחיד** (-v)
3. 0_F · v = 0_V (סקלר אפס כפול כל וקטור = וקטור אפס)
4. α · 0_V = 0_V (כל סקלר כפול וקטור אפס = וקטור אפס)
5. אם α·v = 0_V אז α=0 או v=0_V
6. (-1)·v = -v

**הערה**: במרחב וקטורי **לא מוגדר** כפל בין שני וקטורים!

## 4. תת מרחב (Subspace)

### מה זה תת מרחב?
W ⊆ V הוא תת מרחב אם הוא בעצמו מרחב וקטורי עם אותן פעולות.

### הקריטריון המקוצר (איך בודקים):
W הוא תת מרחב אם ורק אם:
1. W סגור לחיבור (w₁+w₂ ∈ W)
2. W סגור לכפל בסקלר (αw ∈ W)
3. 0_V ∈ W

**או בקיצור עוד יותר**: לכל α₁,α₂∈F ולכל w₁,w₂∈W מתקיים α₁w₁+α₂w₂ ∈ W (סגירות לצירוף ליניארי).

### תת מרחבים טריוויאליים:
לכל מרחב וקטורי יש שני תת מרחבים טריוויאליים:
- V עצמו
- {0_V} — תת מרחב האפס

### משפטים:
- **חיתוך**: W₁ ∩ W₂ תמיד תת מרחב של V
- **איחוד**: W₁ ∪ W₂ **לא** בהכרח תת מרחב (חשוב!)
- **טענה**: אם V = W₁ ∪ W₂ אז W₁=V או W₂=V

## 5. מרחב פולינומים

### מה זה F[x]?
קבוצת כל הפולינומים מעל שדה F:
F[x] = {a₀ + a₁x + a₂x² + ... + aₙxⁿ | n≥0, aᵢ∈F}

### דרגה (degree):
- אם p(x) = a₀+a₁x+...+aₙxⁿ ו-aₙ≠0 → deg(p) = n
- אם כל המקדמים אפס → deg(p) = -∞
- **F[x] הוא מרחב וקטורי מעל F**

### Fₙ[x] — פולינומים מדרגה לכל היותר n:
Fₙ[x] = {p(x) ∈ F[x] | deg(p) ≤ n}

**משפט**: Fₙ[x] הוא תת מרחב של F[x]

### שוויון פולינומים:
p = q אמ"מ deg(p) = deg(q) ולכל i מתקיים aᵢ = bᵢ

### הצבה ושורש:
- **הצבה**: p(α) = a₀ + a₁α + ... + aₙαⁿ
- **שורש**: α הוא שורש של p אם p(α) = 0

## 6. Span (פרישה) — הרצאות 9-10

### צירוף ליניארי:
v הוא **צירוף ליניארי** של v₁,...,vₖ אם קיימים סקלרים α₁,...,αₖ כך ש:
v = α₁v₁ + ... + αₖvₖ

**בעברית**: אפשר "להרכיב" את v מהוקטורים האחרים ע"י כפל בסקלרים וחיבור.

### Span — הגדרה:
span{v₁,...,vₖ} = {α₁v₁+...+αₖvₖ | α₁,...,αₖ∈F}
כלומר — **קבוצת כל הצירופים הליניאריים** של הוקטורים.

span∅ = {0_V}

### למה Span חשוב?
**משפט**: span{v₁,...,vₖ} הוא **תת מרחב** של V!
זה בעצם תת המרחב הכי קטן שמכיל את כל הוקטורים v₁,...,vₖ.

### קבוצה פורשת (spanning set):
S פורשת את V אם span(S) = V, כלומר כל וקטור ב-V אפשר לכתוב כצירוף ליניארי של איברי S.

### נוצר סופית (finitely generated):
מרחב V נוצר סופית אם קיימת קבוצה סופית S כך ש V = span(S).

**דוגמאות**:
- Fⁿ נוצר סופית (e₁,...,eₙ פורשים אותו)
- Fₙ[x] נוצר סופית (1,x,x²,...,xⁿ פורשים אותו)
- F[x] **לא** נוצר סופית!

### משפטים ומסקנות על Span:

**משפט**: כל vᵢ שייך ל-span{v₁,...,vₖ} (כל וקטור הוא חלק מה-span שלו)

**משפט**: אם A⊆B אז span(A) ⊆ span(B)

**מסקנה**: span(A) = span(B) אמ"מ A⊆span(B) וגם B⊆span(A)

**משפט**: v ∈ span{v₁,...,vₖ} ⟺ span{v₁,...,vₖ} = span{v₁,...,vₖ,v}
(אם וקטור כבר ב-span, הוספה שלו לא משנה כלום)

**Span של קבוצה כללית (סופית או אינסופית)**:
span(S) = {α₁v₁+...+αₖvₖ | k∈N, αᵢ∈F, vᵢ∈S}

### פרישה מינימלית:
v₁,...,vₙ פורשים **מינימלית** את V אם:
1. הם פורשים את V
2. הורדה של כל וקטור אחד — כבר לא פורשים

## 7. תלות ליניארית — הרצאות 10-11

### הגדרה:
v₁,...,vₖ **בלתי תלויים ליניארית (בת"ל)** אם:
α₁v₁ + ... + αₖvₖ = 0 ⟹ α₁=...=αₖ=0

כלומר, הדרך **היחידה** לקבל את וקטור האפס מצירוף ליניארי שלהם היא שכל המקדמים יהיו אפס.

v₁,...,vₖ **תלויים ליניארית (ת"ל)** אם:
קיימים α₁,...,αₖ **לא כולם אפס** כך ש α₁v₁+...+αₖvₖ = 0

### איך לחשוב על זה:
- **בת"ל** = אף וקטור לא "מיותר", אף אחד מהם לא צירוף ליניארי של האחרים
- **ת"ל** = יש לפחות וקטור אחד שהוא צירוף ליניארי של האחרים (הוא "מיותר")

### משפטים חשובים:

**משפט**: v₁,...,vₖ ת"ל אמ"מ קיים j כך ש vⱼ ∈ span{v₁,...,vⱼ₋₁}
(אחד מהם "תלוי" בקודמים)

**נוסח שקול**: v₁,...,vₖ בת"ל אמ"מ לכל j: vⱼ ∉ span{v₁,...,vⱼ₋₁}

**הערה**: כשהמקדמים כולם 0 זה נקרא "הצירוף הטריוויאלי". וקטורים בת"ל = הצירוף הטריוויאלי הוא היחיד שנותן אפס.

### מסקנות:
- אם A בת"ל ו-B⊆A → גם B בת"ל
- אם B ת"ל ו-B⊆A → גם A ת"ל
- אם 0_V ∈ A → A ת"ל (תמיד! כי α·0=0 לכל α)

### משפט (שני וקטורים):
v₁,v₂ ∈ V בת"ל אמ"מ אף אחד מהם אינו כפולה סקלרית של השני.

### משפט יחידות הייצוג:
אם v₁,...,vₖ **בת"ל** אז כל v ∈ span{v₁,...,vₖ} ניתן לייצוג **יחיד** כצירוף ליניארי.

**זה סופר חשוב!** כשיש בת"ל + פרישה (=בסיס), הייצוג הוא יחיד — וזה מה שמאפשר לנו לעבוד עם קואורדינטות.

### הגדרה: בת"ל מקסימלית
v₁,...,vₙ בת"ל מקסימלית אם:
1. הם בת"ל
2. לכל v∈V, הקבוצה {v₁,...,vₙ,v} כבר ת"ל
(כלומר, לא ניתן להוסיף עוד וקטור ולהישאר בת"ל)

---

## 8. בסיס (Basis) — הרצאות 11-12

### הגדרה:
v₁,...,vₙ הם **בסיס** של V אם:
1. v₁,...,vₙ **בת"ל**
2. v₁,...,vₙ **פורשים** את V

**הערה**: אם v₁,...,vₙ בסיס, אז כל v∈V ניתן לייצוג כצירוף ליניארי **יחיד**.

### בסיסים סטנדרטיים:

**בסיס סטנדרטי של Fⁿ**:
e₁ = (1,0,...,0), e₂ = (0,1,...,0), ..., eₙ = (0,...,0,1)

**בסיס סטנדרטי של M_{m×n}(F)**:
המטריצות Eᵢⱼ (1 במקום i,j ו-0 בכל השאר)

**בסיס סטנדרטי של Fₙ[x]**:
e₀(x)=1, e₁(x)=x, e₂(x)=x², ..., eₙ(x)=xⁿ

### למת שטיינל (Steinitz Exchange Lemma):
יהי V מ"ו מעל F, יהיו v₁,...,vₖ בסיס ו-w₁,...,wₘ פורשים.
אז: **k ≤ m**

**מסקנה מהלמה**: כל שני בסיסים של V הם באותו גודל! (k = m)

### משפט:
V נוצר סופית ו-V≠{0} ⟹ קיימים v₁,...,vₙ שהם בסיס של V.

---

## 9. מימד (Dimension) — הרצאות 13-18

### הגדרה:
- אם V={0_V} → dim V = 0
- אם V≠{0_V} ונוצר סופית → dim V = מספר הוקטורים בבסיס (לא משנה איזה בסיס!)
- אם V לא נוצר סופית → dim V = ∞

### מסקנות מיידיות:

**מסקנה 1**: אם dim V = n, אז כל n+1 וקטורים או יותר הם **ת"ל**.

**מסקנה 2**: אם dim V = n, אז כל n-1 וקטורים או פחות **לא פורשים** את V.

### משפט השלש חזק (dim = n, יש לנו n וקטורים):
אם dim V = n ויהיו v₁,...,vₙ∈V, אז **שלושת התנאים שקולים**:
1. v₁,...,vₙ **בסיס** של V
2. v₁,...,vₙ **בת"ל**
3. v₁,...,vₙ **פורשים** את V

**זה גאוני!** אם יודעים את המימד, מספיק לבדוק רק אחד מהשלושה!

### משפט ההשלמה לבסיס:
אם dim V = n ויש v₁,...,vₖ בת"ל:
- k ≤ n
- אם k < n, אז קיימים vₖ₊₁,...,vₙ כך ש v₁,...,vₙ **בסיס** של V

(כלומר, תמיד אפשר "להשלים" קבוצה בת"ל לבסיס!)

### מימד של תת מרחב:
- W תת מרחב של V נוצר סופית → W נוצר סופית
- **dim W ≤ dim V**
- **dim W = dim V ⟹ W = V** (אם תת מרחב באותו מימד — הוא המרחב כולו!)

### סכום ישר:

**סכום**: U+W = {u+w | u∈U, w∈W}

**סכום ישר**: U⊕W (כאשר U∩W = {0_V})

**משפט**: U+W תמיד תת מרחב, ומתקיים U∪W ⊆ U+W

**סיכום שרשרת**: {0_V} ⊆ U∩W ⊆ U,W ⊆ U+W ⊆ V

### משפט המימדים הראשון:
dim(U+W) = dim U + dim W - dim(U∩W)

**מסקנה**: אם V = U⊕W אז dim V = dim U + dim W

### משפט (span של איחוד):
U = span{u₁,...,uₖ} ו-W = span{w₁,...,wₗ}
⟹ U+W = span{u₁,...,uₖ,w₁,...,wₗ}

## 10. מטריצות — שוויון, חיבור, כפל בסקלר

### שוויון מטריצות:
A = B אמ"מ [A]ᵢⱼ = [B]ᵢⱼ לכל i,j (כל כניסה שווה)

### חיבור וכפל בסקלר:
- [A+B]ᵢⱼ = [A]ᵢⱼ + [B]ᵢⱼ
- [αA]ᵢⱼ = α·[A]ᵢⱼ

**משפט**: M_{m×n}(F) עם פעולות אלה הוא מרחב וקטורי!

### מערכת הומוגנית ומרחב:
קבוצת הפתרונות של Ax̄=0̄ **אינה** תת מרחב (היא כן! NulA הוא תת מרחב).
קבוצת הפתרונות של Ax̄=b̄ (כש b̄≠0̄) **אינה** תת מרחב!

---

## 11. כפל מטריצות — הרצאות 16-18

### הגדרה:
A∈M_{m×n}(F), B∈M_{n×r}(F) → AB מוגדרת ומסדר m×r.
[AB]ᵢⱼ = Σₖ [A]ᵢₖ·[B]ₖⱼ

**שימו לב**: מספר העמודות של A חייב להיות שווה למספר השורות של B!

### תכונות חשובות:
- **כפל מטריצות אינו קומוטטיבי!** AB ≠ BA בכלל (ייתכן שאחד מוגדר והשני לא)
- **יש מחלקי אפס**: AB=0 לא אומר ש-A=0 או B=0
- **לפעמים AB מוגדר אבל BA לא**
- **אסוציאטיביות**: (AB)C = A(BC) ✓
- **דיסטריביוטיביות**: A(B+C) = AB+AC ו-(A+B)C = AC+BC ✓
- **סקלר**: (αA)B = A(αB) = α(AB) ✓

### קומוטטיביות:
A,B מתחלפות (commute) אם AB = BA

### מטריצת היחידה Iₙ:
[Iₙ]ᵢⱼ = 1 אם i=j, ו-0 אחרת
- I_m · A = A · I_n = A
- מטריצת האפס: O·A = A·O = O

### כפל מטריצה בוקטור — הקשר למערכות ליניאריות:
**משפט**: אם A = (C̄₁ ··· C̄ₙ) (עמודות) אז:
Ax̄ = x₁C̄₁ + x₂C̄₂ + ... + xₙC̄ₙ

כלומר, **Ax̄ הוא צירוף ליניארי של עמודות A!**

### מסקנות מהקשר הזה:
1. למערכת Ax̄=0̄ יש פתרון יחיד אמ"מ עמודות A **בת"ל**
2. אם m<n (פחות משוואות ממשתנים) → Ax̄=0̄ יש יותר מפתרון אחד
3. Ax̄=b̄ יש פתרון אמ"מ b̄ ∈ ColA (הוקטור b̄ שייך למרחב העמודות)
4. AB = A(C̄₁ ··· C̄ᵣ) = (AC̄₁ ··· AC̄ᵣ)

---

## 12. מטריצה הפיכה — הרצאות 16-18

### הגדרות:
- **הפיכה משמאל** (left invertible): קיימת B כך ש BA=Iₙ
- **הפיכה מימין** (right invertible): קיימת C כך ש AC=Iₙ
- **הפיכה** (invertible): הפיכה משמאל וגם מימין

### מסקנות:
- הופכית משמאל ומימין **לא** בהכרח מוגדרות באופן יחיד
- אבל: אם A הפיכה משמאל **וגם** מימין → הם **שווים**! (B = C = A⁻¹)

**הגדרה**: A∈M_n(F) הפיכה אם קיימת B **יחידה** כך ש AB=BA=Iₙ. B=A⁻¹.

### משפט: A∈M_n(F) הפיכה אמ"מ...
- rank A = n
- עמודות A בת"ל
- עמודות A מהוות בסיס ל-Fⁿ
- Ax̄=b̄ יש פתרון יחיד לכל b̄
- A ניתנת לייצוג כמכפלת מטריצות אלמנטריות
- det A ≠ 0

### תכונות ההופכית:
- (Iₙ)⁻¹ = Iₙ
- (A⁻¹)⁻¹ = A
- (AB)⁻¹ = B⁻¹·A⁻¹ (**שימו לב להיפוך הסדר!**)
- (A₁···Aₖ)⁻¹ = Aₖ⁻¹···A₁⁻¹

### מציאת הופכית בפועל (אלגוריתם ההיפוך):
בונים (A | Iₙ) ומדרגים. אם A הפיכה, נקבל (Iₙ | A⁻¹).
E_k···E₁·A = C (הצורה הקנונית).
אם C=Iₙ → A הפיכה ו-A⁻¹ = Eₖ···E₁.

### טענת עזר:
A∈M_n(F) — אם ל-A שורות אפסים, היא **לא הפיכה**.

### מטריצות אלמנטריות:
E∈M_n(F) היא **מטריצה אלמנטרית** אם מתקבלת מ-Iₙ ע"י פעולת שורה אלמנטרית אחת.

שלושה סוגים:
1. Iₙ עם כפל שורה i ב-c≠0 → E
2. Iₙ עם החלפת שורות i↔j → E
3. Iₙ עם הוספת α כפולת שורה j לשורה i → E

**כל מטריצה אלמנטרית הפיכה!** וההופכית שלה גם אלמנטרית.

### הקשר בין שקילות שורה והפיכות:
- A,B שקולות שורה ⟹ A הפיכה אמ"מ B הפיכה
- EA=B (E אלמנטרית) מייצג פעולת שורה אחת על A

---

## 13. מטריצה משוכלפת (Transpose)

### הגדרה:
Aᵗ מסדר n×m מוגדרת ע"י [Aᵗ]ᵢⱼ = [A]ⱼᵢ (מחליפים שורות ועמודות)

### תכונות:
- (A+B)ᵗ = Aᵗ + Bᵗ
- (αA)ᵗ = α·Aᵗ
- (Aᵗ)ᵗ = A
- **(AB)ᵗ = Bᵗ·Aᵗ** (שימו לב להיפוך!)
- A הפיכה ⟺ Aᵗ הפיכה, ואז (A⁻¹)ᵗ = (Aᵗ)⁻¹

### מטריצה סימטרית ואנטי-סימטרית:
- **סימטרית**: A = Aᵗ
- **אנטי-סימטרית**: A = -Aᵗ

**משפט**: S (סימטריות) ו-AS (אנטי-סימטריות) הם תתי מרחבים של Mₙ(F).
אם 1+1≠0 אז Mₙ(F) = S ⊕ AS

## 14. מרחב אפס (NulA), מרחב עמודות (ColA), מרחב שורות (RowA)

### מרחב האפס:
NulA = {x̄ ∈ Fⁿ | Ax̄ = 0̄}
הוא **תת מרחב** של Fⁿ.

**טיפ**: מימד NulA = מספר המשתנים החופשיים. כדי למצוא בסיס — מדרגים ומציבים 1 בכל פעם במשתנה חופשי אחר.

### מרחב העמודות:
ColA = span{C̄₁,...,C̄ₙ} ⊆ Fᵐ

### מרחב השורות:
RowA = span{R̄₁ᵗ,...,R̄ₘᵗ} ⊆ Fⁿ. גם שווה ל-ColAᵗ.

### הקשרים:
- **Ax̄=b̄ יש פתרון אמ"מ b̄∈ColA**
- **A הפיכה ⟺ ColA = Fⁿ ⟺ RowA = Fⁿ ⟺ rankA = n**

### דרגת עמודות ודרגת שורות:
- col-rank A = dim ColA
- row-rank A = dim RowA

**משפט**: **dimColA = dimRowA = rankA** (דרגת העמודות שווה לדרגת השורות!)

### הערות חשובות:
- הבסיס של RowA הם השורות שאינן אפס בצורה המדורגת (הקנונית)
- A הפיכה ⟺ A ניתנת לייצוג כמכפלת מטריצות אלמנטריות
- A,B שקולות שורה → RowA = RowB

### הקשר בין Ax̄=b̄ ו-Ax̄=0̄:
**משפט**: אם G = {x̄∈Fⁿ | Ax̄=b̄} ≠ ∅ ויהי x̄₀∈G, אז:
G = {ū + x̄₀ | ū ∈ NulA}

**בעברית**: קבוצת הפתרונות של המערכת הלא-הומוגנית = פתרון פרטי + כל הפתרונות של ההומוגנית.

### משפט הדרגה (Rank-Nullity):
**rankA + dimNulA = n** (מספר העמודות)

## 15. דטרמיננטות — הרצאות 20-23

### הרעיון:
פונקציה Δ: Mₙ(F)→F שנותנת **מספר** לכל מטריצה ריבועית.

### לינאריות בשורה i:
Δ **ליניארית בשורה i** אם כשמשנים רק את שורה i (והשאר נשאר) — הפונקציה מתנהגת ליניארית.

### מולטי-ליניארית:
Δ **מולטי-ליניארית** אם היא ליניארית בכל שורה (כל אחת בנפרד).

### מתחלפת (alternating):
Δ **מתחלפת** אם A עם שתי שורות זהות → Δ(A) = 0.

### דטרמיננטה — הגדרה:
פונקציה שהיא:
1. מולטי-לינארית
2. מתחלפת
3. Δ(Iₙ) = 1

### משפט (*) — הקשר לפעולות שורה:
אם B מתקבלת מ-A ע"י פעולת שורה אלמנטרית:
1. Rᵢ→cRᵢ (c≠0): Δ(B) = c·Δ(A)
2. Rᵢ↔Rⱼ: Δ(B) = -Δ(A)
3. Rᵢ→Rᵢ+αRⱼ: Δ(B) = Δ(A)

### מסקנות מהמשפט:
- אם B מתקבלת מ-A ע"י הכפלת שורה ב-c → Δ(B) = c·Δ(A) ⟹ Δ(A)=0 או Δ(B)=0
- אם A,B שקולות שורה ו-Δ(A)=0 → Δ(B)=0

### משפט: A לא הפיכה ⟺ Δ(A) = 0
(הדטרמיננטה מזהה הפיכות!)

### יחידות פונקצי הדטרמיננטה:
**משפט חשוב 3**: לכל n∈N קיימת פונקצית דטרמיננטה **יחידה**!

### det — ההגדרה הרקורסיבית:
- n=1: det(a) = a
- n=2: det((a,b),(c,d)) = ad - bc
- n≥2: detA = Σᵢ (-1)^(i+1) · [A]ᵢ₁ · Mᵢ₁⁽ᴬ⁾ (פיתוח לפי עמודה 1)

כאשר Mᵢⱼ⁽ᴬ⁾ הוא **המינור** — הדטרמיננטה של המטריצה שמתקבלת מחיקת שורה i ועמודה j.

### פיתוח לפי עמודה j:
detA = Σᵢ (-1)^(i+j) · [A]ᵢⱼ · Mᵢⱼ⁽ᴬ⁾

### פיתוח לפי שורה i:
detA = Σⱼ (-1)^(i+j) · [A]ᵢⱼ · Mᵢⱼ⁽ᴬ⁾

### דטרמיננטה של מטריצה משולשית:
**משפט**: אם A משולשית (עליונה או תחתונה), אז:
detA = [A]₁₁ · [A]₂₂ · ... · [A]ₙₙ (מכפלת האלכסון!)

### נוסחה ל-3×3 (דרך לזכור):
detA = aei + bfg + cdh - afh - bdi - ceg

### תכונות det:

**משפט חשוב 1**: det(AB) = det(A)·det(B)

**למה 1**: det(EA) = det(E)·det(A) (E אלמנטרית)

**למה 2**: det(Eᵗ) = det(E) (E אלמנטרית)

**משפט חשוב 2**: det(Aᵗ) = det(A) (לכל A!)

**מסקנה**: A הפיכה → det(A⁻¹) = (det A)⁻¹

**מסקנה**: A הפיכה ⟺ Aᵗ הפיכה

**מסקנה**: עמודות Aᵗ מהוות בסיס ל-Fⁿ ⟺ A הפיכה

**משפט**: det מולטי-ליניארית (כל n)

**משפט**: det מתחלפת (כל n≥2)

### מטריצת בלוקים:
k = ((A, C), (O, B)) כאשר A∈Mₙ(F), B∈Mₘ(F)
**אם** C=O (בלוקים משולשית עליונה) או D=O (תחתונה):
detk = detA · detB

## 16. סיכום קשרים חשובים ומסקנות כלליות

### A∈Mₙ(F) — כל התנאים הבאים **שקולים**:
- A הפיכה
- rank A = n
- det A ≠ 0
- עמודות A בת"ל
- עמודות A בסיס ל-Fⁿ
- שורות A בת"ל
- שורות A בסיס ל-Fⁿ
- Ax̄=b̄ יש פתרון יחיד לכל b̄

- NulA = {0̄}

rankA + dim NulA = n
