הרצאה 20 - נגזרת פונקציה הפוכה וכלל השרשרת

20 משפט 1: נגזרת פונקציה הפוכה
(f⁻¹)'(y₀) = 1 / f'(x₀)
אם f הופכת x ל-y בקצב מסוים, אז f⁻¹ הופכת y חזרה ל-x בקצב ההפוך (1 חלקי). כמו: אם נוסע 60 קמ"ש, אז הזמן לקילומטר הוא 1/60 שעה.
20 משפט 2: (eˣ)' = eˣ — הפונקציה המושלמת
(eˣ)' = eˣ   לכל x∈ℝ
eˣ היא הפונקציה היחידה שקצב הגדילה שלה שווה בדיוק לגודל שלה. מוכח דרך: ln היא ההפוכה של eˣ, ומשפט 1.
20 משפט 3: (arctg x)' = 1/(1+x²)
(arctg x)' = 1/(1+x²)
arctg ההפוכה של tg. משפט 1 על tg (נגזרתה 1/cos²x), ואז זהות 1+tg²x = 1/cos²x נותנת ביטוי נקי ב-y.
20 משפט 4: (arcsin x)' = 1/√(1-x²)
(arcsin x)' = 1/√(1-x²)   לכל -1 < x < 1
ההפוכה של sin. לפי משפט 1 מקבלים 1/cos, ואז cos = √(1-sin²x) = √(1-y²).
20 משפט 5: (arccos x)' = -1/√(1-x²)
(arccos x)' = -1/√(1-x²)   לכל -1 < x < 1
כמו arcsin רק עם מינוס - כי cos יורד, אז ההפוכה שלו "הפוכה בכיוון".
20 משפט 6: כלל השרשרת (Chain Rule)
(g∘f)'(x₀) = g'(f(x₀)) · f'(x₀)
פונקציה בתוך פונקציה - גוזרים "מבחוץ פנימה" וכופלים. [sin(x²)]' = cos(x²)·2x. שינוי קטן ב-x עובר דרך f ואז דרך g - הקצבים מוכפלים.
20 הגדרה: גזירה בקטע [a,b]
גזירה ב-(a,b) + נגזרות חד-צדדיות בקצוות
בקצוות אי אפשר לגשת משני הצדדים, אז מסתפקים בגבול חד-צדדי: f'₊(a) מימין ו-f'₋(b) משמאל.

הרצאה 21 - נגזרות חזקות, פרמה, רול, לגרנז'

21 משפט 1: (xᵅ)' = α·xᵅ⁻¹
(xᵅ)' = α · xᵅ⁻¹   לכל x>0, α∈ℝ
"מורידים חזקה וגורעים 1" עובד לכל α ממשי! מוכח ע"י xᵅ = eᵅˡⁿˣ + כלל שרשרת.
21 משפט 2: (aˣ)' = ln(a)·aˣ
(aˣ)' = ln(a) · aˣ
כמו eˣ עם "תיקון" ln(a). מוכח דרך aˣ = eˣˡⁿᵃ + כלל שרשרת.
21 משפט 3: (logₐx)' = 1/(ln(a)·x)
(logₐx)' = 1 / (ln(a) · x)
כמו ln(x)' = 1/x, רק חלקי ln(a). כי logₐx = lnx/lna.
21 משפט 4: פרמה (Fermat)
נק' קיצון פנימית + f גזירה → f'(x₀) = 0
במקס/מין מקומי המשיק אופקי. משמאל עלתה (≥0), מימין ירדה (≤0), אז ב-x₀ חייב 0. זהירות: ההפך לא נכון! (x³ ב-0).
21 משפט 5: רול (Rolle)
רציפה [a,b], גזירה (a,b), f(a)=f(b) → ∃c: f'(c)=0
יצאת מגובה וחזרת לאותו גובה - חייב מקום עם משיק אופקי. פרמה + ויירשטראוס ביחד.
21 מסקנה מרול: f'≠0 → חח"ע
f גזירה ב-[a,b], f'(x)≠0 לכל x → f חד-חד-ערכית
אם הנגזרת לא מתאפסת, הפונקציה לא חוזרת לערך קודם (אחרת רול היה נותן f'(c)=0).
21 משפט 6: לגרנז' (Lagrange / MVT)
רציפה [a,b], גזירה (a,b) → ∃c: f'(c) = [f(b)-f(a)]/(b-a)
בין שתי נקודות יש מקום שהמשיק מקביל לקו שמחבר אותן (שיפוע רגעי = שיפוע ממוצע). הוכחה: מחסרים קו ישר ומפעילים רול.
21 מסקנה 1+2: סימן הנגזרת → מונוטוניות
f'≥0 → עולה | f'≤0 → יורדת | f'=0 → קבועה | f'>0 → עולה ממש
הכלי המרכזי לחקירת פונקציה. מוכח ישירות מלגרנז': f(x₂)-f(x₁) = f'(c)·(x₂-x₁), הסימן נקבע ע"י f'(c).

הרצאה 22 - יישומים ואי-שוויונות

22 תרגיל 1: eˣ ≥ 1+x
eˣ ≥ 1 + x   לכל x ≥ 0
האקספוננט תמיד מעל המשיק שלו ב-0. מוכח: f(x)=eˣ-1-x, נגזרתה eˣ-1≥0 ל-x≥0, אז f עולה מ-f(0)=0.
22 מסקנה 3: |sinx-siny| ≤ |x-y| וגם |cosx-cosy| ≤ |x-y|
|sin x - sin y| ≤ |x-y|   ,   |cos x - cos y| ≤ |x-y|
sin/cos לא משתנים מהר יותר מ-x (כי |נגזרת| ≤ 1). מלגרנז': ההפרש = cos(c)·(x-y), ו-|cos c|≤1.
22 מסקנה 4: x/(1+x) ≤ ln(1+x) ≤ x
x/(1+x) ≤ ln(1+x) ≤ x   לכל x > -1
ln(1+x) כלוא בין שני קירובים. מלגרנז' על [0,x]: ln(1+x)/x = 1/(1+c), וחוסמים את c בין 0 ל-x.

תמונה גדולה - הזרימה הלוגית

הגדרת הנגזרת → נגזרת הפוכה → מאפשרת לגזור eˣ, arctg, arcsin, arccos
כלל השרשרת + הנ"ל → מאפשרים לגזור xᵅ, aˣ, logₐx ופונקציות מורכבות

פרמה (נגזרת=0 בקיצון) + ויירשטראוס (קיום מקס/מין) → רול (f(a)=f(b) → f'(c)=0)
רול → לגרנז' (שיפוע ממוצע = שיפוע רגעי) → מסקנות מונוטוניות → אי-שוויונות שימושיים
