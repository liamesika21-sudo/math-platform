אינפי 1 — מועד ב' 29/4

חזק: גזירות + גבולות בינוני: inf/sup הגדרות חלש: IVT + פונקציות עזר

הוכחות לשינון בעל פה חובה

  • משפט רול (Rolle's Theorem)
  • משפט לגראנז' (Mean Value Theorem)
  • משפט קושי (Cauchy's MVT) — כולל בניית \(h(x) = f(x) - \alpha g(x)\)
  • הרכבת גבולות — הופיע ב-2023 מועד ב'!
  • צפיפות Q ב-R — הופיע כמעט כל שנה
  • אפיון inf: \(s = \inf(A) \iff \forall\varepsilon>0\;\exists a\in A: a < s+\varepsilon\)
  • גזירות פונקציה הופכית — הופיע 2022 מועד ב', 2025 מועד ב'

הגדרות לשינון חובה

  • גבול של פונקציה (\(\varepsilon\)-\(\delta\))
  • \(f\) גזירה ב-\(x_0\) (עם \(\varepsilon, \delta\))
  • \(A\) חסומה מלעיל / מלרע
  • \(\inf(A) = s\)
  • \(f\) רציפה ב-\(x_0\)
  • \(f\) עולה / יורדת ממש
  • לא חסומה (בלי שלילה!)
  • גבול אינסוף / מינוס אינסוף

תבניות שאלות לתרגל עדיפות גבוהה

  • IVT עם פונקציית עזר (כמו 3.2) — בנה g, בדוק סימנים, הפעל IVT
  • לגראנז' על f'' (כמו 4.1) — הפעל לגראנז' פעמיים
  • הוכח/הפרך (T/F) — דוגמאות נגדיות + שימוש במשפטים
  • הוכחות עם צפיפות Q
  • אי-שוויונות עם פונקציות עזר (כמו \(\cos x \geq 1-\frac{x^2}{2}\))

בדידה — מועד ב' 7/5

TOP 5 נושאים שחוזרים בשאלות פתוחות חובה

  • 1. יחסי שקילות + מחלקות (70% מהמבחנים!) — למצוא מחלקות, לחשב |A/R|
  • 2. אינדוקציה (40%) — רגילה + חזקה, פירוק למקרים
  • 3. Hall's Marriage / Stable Matching (30%) — הופיע 2023B, 2022B, 2019B
  • 4. סדר חלקי + האסה (30%) — מינימלי, מקסימלי, שרשראות, אנטי-שרשראות
  • 5. פונקציות חח"ע/על עם הרכבה — הוכח שמרכיב חח"ע/על

משפטים לזכור עדיפות גבוהה

  • Hall's Marriage Theorem: matching שלם קיים אמ"מ \(\forall S: |N(S)| \geq |S|\)
  • שובך היונים: \(n+1\) יונים ב-\(n\) שובכים → לפחות 2 באותו שובך
  • עקרון הכלילה-הדחה: \(|A \cup B| = |A| + |B| - |A \cap B|\)
  • Binomial Theorem + זהויות בינום (ונדרמונד, בינום שלילי)
  • כל יחס שקילות מגדיר חלוקה (ולהפך)

שאלות אמריקאיות — נושאים חמים בינוני

  • \(p \to q \equiv \neg p \lor q\) — שקילויות לוגיות
  • CNF / DNF — המרה + זיהוי
  • קומבינטוריקה עם אילוצים (ישיבה בשורה, חלוקה לקבוצות)
  • ספירה: סדר חשוב / סדר לא חשוב, עם/בלי חזרות
  • פרדיקטים: שלילת כמתים מקוננים

לינארית 1 — מועד ב' 15/5

6 הוכחות עם הסיכוי הגבוה ביותר חובה — שינון בעל פה

  • 1. Rank-Nullity: \(\text{rank}(A) + \dim(\text{Nul}\,A) = n\) — הופיע כמעט כל שנה
  • 2. טבלת הפיכות: A הפיכה ⟺ rank=n ⟺ det≠0 ⟺ עמודות בת"ל ⟺ ...
  • 3. שלושת המשפטים: dim(V)=n → n וקטורים: בסיס ⟺ בת"ל ⟺ פורשים
  • 4. det(AB) = det(A)·det(B) — הוכחה עם 2 מקרים (הפיכה / לא הפיכה)
  • 5. Row rank = Column rank
  • 6. Steinitz + השלמה לבסיס

נושאים לחישוב עדיפות גבוהה

  • דירוג מטריצות + מציאת rank
  • מציאת בסיס ל-Col(A), Row(A), Nul(A)
  • חישוב דטרמיננטות (פיתוח לפי שורה/עמודה, תכונות)
  • מציאת מטריצה הפוכה
  • בדיקת תלות/אי-תלות ליניארית
  • מציאת בסיס לחיתוך/סכום מרחבים
  • מטריצה מייצגת של העתקה ליניארית
  • שינוי בסיס + מטריצת מעבר

הגדרות קריטיות חשוב

  • מרחב וקטורי, תת-מרחב
  • תלות/אי-תלות ליניארית
  • בסיס, ממד
  • rank של מטריצה
  • העתקה ליניארית, גרעין, תמונה
  • דטרמיננטה (הגדרה רקורסיבית)
