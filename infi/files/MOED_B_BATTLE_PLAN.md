## STEP 3 — 10 MOST PROBABLE THEOREMS TO MEMORIZE

### THEOREM 1: Differentiability implies Continuity
**Statement:** If f is differentiable at x₀, then f is continuous at x₀.

**Proof:**
f(x) - f(x₀) = [(f(x) - f(x₀))/(x - x₀)] · (x - x₀)

As x → x₀:
- (f(x) - f(x₀))/(x - x₀) → f'(x₀) (exists by assumption)
- (x - x₀) → 0

By algebra of limits: f(x) - f(x₀) → f'(x₀) · 0 = 0
Therefore f(x) → f(x₀), so f is continuous at x₀. ∎

**Why it will appear:** Not tested in Moed A 2026. Tested in Simulation 2026 Q1.1.1. It's the most "free" theorem proof — short, clean, testable.

**Source:** Lecture 11; Simulation 2026 Q1.1.1

### THEOREM 2: Rolle's Theorem
**Statement:** If f is continuous on [a,b], differentiable on (a,b), and f(a)=f(b), then there exists c ∈ (a,b) such that f'(c)=0.

**Proof:**
Since f is continuous on [a,b], by Weierstrass theorem, f attains maximum M and minimum m on [a,b].

**Case 1:** M = m. Then f is constant on [a,b], so f'(c)=0 for every c ∈ (a,b). ∎

**Case 2:** M ≠ m. Since f(a)=f(b), at least one of M, m is attained at an interior point c ∈ (a,b). Since f is differentiable at c and c is a local extremum, by Fermat's theorem: f'(c) = 0. ∎

**Why it will appear:** Core mechanism for countless questions. Even if not asked directly, the proof structure appears in auxiliary function problems.

**Source:** Lecture 17

### THEOREM 3: Lagrange MVT
**Statement:** If f is continuous on [a,b] and differentiable on (a,b), then ∃c ∈ (a,b) such that f'(c) = (f(b)-f(a))/(b-a).

**Proof:**
Define g(x) = f(x) - f(a) - [(f(b)-f(a))/(b-a)]·(x-a).
- g is continuous on [a,b] (as combination of continuous functions)
- g is differentiable on (a,b)
- g(a) = 0
- g(b) = f(b) - f(a) - (f(b)-f(a)) = 0

By Rolle's theorem: ∃c ∈ (a,b) such that g'(c) = 0.
g'(c) = f'(c) - (f(b)-f(a))/(b-a) = 0
Therefore f'(c) = (f(b)-f(a))/(b-a). ∎

**Why it will appear:** THE most used theorem in the course. Even if the proof isn't asked, the application is guaranteed.

**Source:** Lecture 17; appears in 70% of all exams

### THEOREM 4: Cauchy MVT
**Statement:** If f,g continuous on [a,b], differentiable on (a,b), and g'(x) ≠ 0 for all x ∈ (a,b), then g(b) ≠ g(a) and ∃c ∈ (a,b) such that f'(c)/g'(c) = (f(b)-f(a))/(g(b)-g(a)).

**Proof (step by step):**

**Step 1: g(b) ≠ g(a).** Suppose g(b)=g(a). By Rolle ∃c with g'(c)=0. Contradiction with g'(x)≠0.

**Step 2:** Define h(x) = f(x) - f(a) - [(f(b)-f(a))/(g(b)-g(a))]·(g(x)-g(a)).
- h continuous on [a,b] ✓
- h differentiable on (a,b) ✓
- h(a) = 0 ✓
- h(b) = f(b)-f(a) - [(f(b)-f(a))/(g(b)-g(a))]·(g(b)-g(a)) = 0 ✓

By Rolle: ∃c ∈ (a,b) with h'(c)=0.
h'(c) = f'(c) - [(f(b)-f(a))/(g(b)-g(a))]·g'(c) = 0
f'(c)/g'(c) = (f(b)-f(a))/(g(b)-g(a)). ∎

**Why it will appear:** You got 0/23 on this in Moed A. Even if the exact same proof isn't asked, the TECHNIQUE of building auxiliary functions from Rolle is CRITICAL. The pattern h(a)=h(b)=0 → apply Rolle is the single most important proof technique in this course.

### THEOREM 6: ε-property of sup(A)
**Statement:** Let A ≠ ∅ be bounded above, s an upper bound. Then s = sup(A) iff ∀ε > 0 ∃a ∈ A : a > s - ε.

**Proof (⇒):** Suppose s = sup(A). Let ε > 0. Then s - ε < s, so s - ε is not an upper bound of A. Therefore ∃a ∈ A with a > s - ε. ✓

**Proof (⇐):** Suppose s is an upper bound and the ε-property holds. Let M be any upper bound of A. ATC M < s. Set ε = s - M > 0. By ε-property, ∃a ∈ A with a > s - ε = M. But M is an upper bound, so a ≤ M. Contradiction. So s ≤ M for every upper bound M, meaning s = sup(A). ∎

**Why it will appear:** Moed A tested the inf version. Moed B will very likely test sup or a variation. This is the workhorse tool for all sup/inf problems.

### TOPIC 1: Bounds / Inf / Sup (Lectures 1-5)

**Key Definitions YOU MUST write perfectly:**
- **Bounded above:** ∃M ∈ R ∀a ∈ A : a ≤ M
- **Bounded below:** ∃m ∈ R ∀a ∈ A : a ≥ m
- **sup(A) = s:** (1) s is an upper bound of A, (2) for every upper bound M of A: s ≤ M
- **inf(A) = s:** (1) s is a lower bound of A, (2) for every lower bound m of A: m ≤ s
- **Maximum:** max(A) = s if s ∈ A and s = sup(A)
- **Archimedean property:** ∀M ∈ R ∃n ∈ N : n > M

**The ε-property (YOUR WEAKNESS — you lost 7 points on this):**
- sup version: s = sup(A) ⟺ s is upper bound AND ∀ε>0 ∃a∈A: a > s-ε
- inf version: s = inf(A) ⟺ s is lower bound AND ∀ε>0 ∃a∈A: a < s+ε
- **CRITICAL:** When using ε-property, you must CHOOSE ε wisely. The typical move: "Let ε = [something involving the quantity you want to contradict]"

**Common exam patterns:**
1. "Prove inf(B) ≤ inf(A)" when B ⊇ A → direct from definition
2. "Compute sup(B)" where B = {f(a) : a ∈ A} → find candidate, prove upper bound, prove ε-property
3. "A has no maximum" → prove ∀a∈A ∃b∈A: b>a (usually via continuity)
4. inf(A-B) = inf(A) - sup(B) → use ε-property for both, combine with triangle

**Traps:**
- Forgetting to prove A ≠ ∅ before using sup/inf
- Forgetting to prove A is bounded before claiming sup/inf exists
- Confusing sup with max (sup need not be in A)

### TOPIC 3: Continuity (Lectures 6, 15-16)

**Key Definitions:**
- **f continuous at x₀:** lim(x→x₀) f(x) = f(x₀)
  Equivalently: ∀ε>0 ∃δ>0 : |x-x₀| < δ ⟹ |f(x)-f(x₀)| < ε
  (Note: no "0 <" before |x-x₀| — this is different from limit definition!)
- **f continuous on [a,b]:** f continuous at every point of [a,b] (with one-sided at endpoints)

**IVT — the #1 tool for existence proofs:**
If f continuous on [a,b] and k is between f(a) and f(b), then ∃c ∈ (a,b) with f(c)=k.

**How to use IVT in exams:**
1. Define f(x) = [the expression you need to show has a root]
2. Show f is continuous on some [a,b]
3. Compute f(a) and f(b) — show they have opposite signs (or k is between them)
4. Apply IVT

**Example from B2024 Q2(b):** Prove ∃x with sin(x)+e^x+x = 3.
- f(x) = sin(x)+e^x+x
- f(0) = 0+1+0 = 1 < 3
- f(1) = sin(1)+e+1 > 3
- By IVT, ∃x ∈ (0,1) with f(x)=3. ∎

**The auxiliary function trick with IVT (YOUR WEAKNESS from Q3.2):**
When asked "prove ∃c with f(c+h)=f(c)":
- Define g(x) = f(x+h) - f(x)
- Show g is continuous
- Find a,b where g(a) and g(b) have different signs
- Apply IVT to get g(c)=0, i.e., f(c+h)=f(c)

**From B2023 Q2(b):** Prove ∃c ∈ [1,2] with f(c)=f(c-1).
- Define g(x) = f(x) - f(x-1) for x ∈ [1,2]

- So g(1) and g(2) have opposite signs (or one is 0)

**Weierstrass:** f continuous on [a,b] ⟹ f attains max and min.
- Used in: "A={x: f(x)≠0} has no maximum" (B2025 Q4.1)

### TOPIC 4: Derivatives (Lectures 11-13, 17)

**Key Definitions:**
- **f differentiable at x₀:** lim(h→0) [f(x₀+h)-f(x₀)]/h exists
  Equivalently: ∃L ∈ R ∀ε>0 ∃δ>0 : 0 < |x-x₀| < δ ⟹ |[f(x)-f(x₀)]/(x-x₀) - L| < ε

**The Big Three: Rolle → MVT → Cauchy MVT**

These are all the SAME idea: build auxiliary function, apply Rolle.

| Theorem | Auxiliary function | Conditions |
|---------|-------------------|------------|
| Rolle | f itself | f(a)=f(b) |
| MVT | g(x) = f(x) - [(f(b)-f(a))/(b-a)]·(x-a) | none extra |
| Cauchy | h(x) = f(x) - [(f(b)-f(a))/(g(b)-g(a))]·g(x) | g'(x)≠0 |

**YOUR STRONGEST MOVE: MVT applied twice (perfect score on Q4.1)**
Pattern:
1. Apply MVT on [a,c] → get f'(c₁)
2. Apply MVT on [c,b] → get f'(c₂)
3. Apply MVT on [c₁,c₂] to f' → get f''(c₃)
4. Compute the bound

**Inequality via auxiliary function (YOU MUST LEARN THIS):**
To prove f(x) ≥ g(x) for all x ≥ 0:
1. Define h(x) = f(x) - g(x)
2. Show h(0) ≥ 0
3. Show h'(x) ≥ 0 for all x ≥ 0
4. Conclude h is increasing, so h(x) ≥ h(0) ≥ 0

**Example: cos(x) ≥ 1 - x²/2 (Moed A Q5.1 — you skipped this!)**
Define h(x) = cos(x) - 1 + x²/2.
h(0) = 0.
h'(x) = -sin(x) + x.
Need to show h'(x) ≥ 0 for x ≥ 0, i.e., x ≥ sin(x).
Define p(x) = x - sin(x). p(0)=0, p'(x) = 1-cos(x) ≥ 0.
So p(x) ≥ 0, meaning x ≥ sin(x), meaning h'(x) ≥ 0.
So h is increasing on [0,∞), h(x) ≥ h(0) = 0. ∎

**Lipschitz from bounded derivative:**
If |f'(x)| ≤ M for all x, then |f(x)-f(y)| ≤ M|x-y|.
Proof: By MVT, f(x)-f(y) = f'(c)(x-y) for some c. So |f(x)-f(y)| = |f'(c)||x-y| ≤ M|x-y|. ∎

## STEP 7 — HOW TO THINK IN THE EXAM

### First 10 minutes: READ ALL 5 QUESTIONS

Do NOT start writing immediately. Read all 5 questions and classify each one:

| If you see... | It's probably... | First move |
|--------------|-----------------|-----------|
| "Define..." with ε,δ | Free points definition | Write it immediately, earn the points |
| "Prove f is irrational" | Irrationality by contradiction | "ATC f(x) = p/q ∈ Q..." |
| "Prove ∃c such that..." | IVT or Rolle | "Define g(x) = ..." |
| "Prove f(x) ≥ g(x)" | Auxiliary function + derivative | "Define h(x) = f(x)-g(x). h(0)=..." |
| "sup/inf of B = ..." | ε-property | "We show s is upper bound. Then ε-property..." |
| f(a), f(b), f(c) values given, prove f'' bound | MVT twice | "By MVT on [a,b]: f'(c₁)=..." |
| "f(q) ≤ g(q) for q ∈ Q" | Density of Q | "Let x ∈ R. By density, ∃qₙ→x..." |
| "Prove or disprove" | Think counterexample FIRST | Is there a simple function that breaks this? |

### Question selection strategy:
1. **Always answer the definition question** — even 2 points is free
2. **Always answer your strongest type** — for you, that's MVT-twice / algebraic proofs
3. **Skip the question with an unfamiliar trick** — better to get 25+25+25+25 on 4 than 20+15+15+10 on 4
4. **If stuck for more than 8 minutes on a sub-question** — write what you know (theorem statement, conditions) and move on. Partial credit exists.

### Time allocation:
- 0:00-0:10 — Read all questions, choose 4
- 0:10-0:50 — Question 1 (your easiest)
- 0:50-1:30 — Question 2
- 1:30-2:10 — Question 3
- 2:10-2:50 — Question 4
- 2:50-3:00 — Review, fix small errors, check conditions

### CRITICAL MISTAKES TO AVOID (from your Moed A):

1. **NEVER write "ברור ש-" (it's clear that).** The grader deducted points for this in Q4.2. Always justify.
2. **ALWAYS state theorem conditions before using it.** "Since f is continuous on [a,b] and differentiable on (a,b), by MVT..."
3. **ALWAYS verify non-zero before dividing.** In Q1.2 Cauchy MVT, you must FIRST prove g(b)≠g(a).
4. **When building auxiliary functions, VERIFY both endpoints.** Write h(a)=0, h(b)=0 explicitly.
5. **Don't confuse "not bounded above" with "not bounded below"** — this cost you 2 points in Q3.1.
6. **In density arguments, use SEQUENCES.** Take qₙ→x with qₙ ∈ Q, then use continuity.
7. **For "prove or disprove": try counterexample first.** Simple functions: constant, sign(x), x², |x|, 1/x, sin(1/x).
